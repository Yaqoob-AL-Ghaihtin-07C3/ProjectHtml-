ما هي لغة CSS

كلمة CSS هي اختصار لجملة Cascading Style Sheet، وهي لغة تستخدم في تصميم صفحات الويب. من خلالها يمكنك تغيير ألوان العناصر، وتحديد مواقعها وأحجامها، وإضافة مؤثرات لها، وجعل صفحات الويب متجاوبة مع مختلف أحجام الشاشات لكي تظهر بشكل ملائم للمستخدم، بالإضافة إلى الكثير من الأمور الرائعة التي ستتعلمها خطوة بخطوة في هذه الدورة.

بشكل عام، CSS تعتبر لغة تصميم (Styling Language) وليس لغة برمجة (Programming Language)، فهي تستخدم في تصميم صفحات الويب التي يتم بناؤها بواسطة HTML أو الصفحات التي يتم بناؤها بواسطة XML.

إصدارات لغة CSS

يتم تطوير لغة CSS باستمرار بدءاً من ظهورها أول مرة عام 1996، والمجموعة المسؤولة عن تطويرها تدعى World Wide Web Consortium أو اختصاراً W3C. يوجد ثلاث إصدارات رئيسية من لغة CSS، والفكرة من هذه الإصدارات ليست تغيير طريقة التعامل مع اللغة، بل إضافة ميزات جديدة عليها.

  • الإصدار الأول (CSS1): فصل كود التصميم عن كود الصفحة الذي يتم بناؤها بواسطة HTML.
  • الإصدار الثاني (CSS2): أضيف فيه خصائص جديدة للتحكم بأماكن العناصر والخطوط وتأثيرات النصوص.
  • الإصدار الثالث (CSS3): أضيفت فيه خصائص مثل Media Queries لجعل التصميم متجاوباً، والتحكم بالرسوم المتحركة والتدرجات اللونية.

عند التعلم، لا تقلق بشأن الإصدار الذي تتعلمه، فأنت في العادة تستخدم خصائص من جميع الإصدارات معاً في أي مشروع.

فوائد لغة CSS

  • توفير الوقت: يمكنك وضع كود التصميم في ملف CSS واستخدامه في أي صفحة تريد.
  • سرعة التحميل: المتصفح يقوم بتحميل ملف CSS مرة واحدة ويخزنه، مما يقلل من وقت تحميل الصفحات اللاحقة.
  • سهولة التعديل: بمجرد التعديل على ملف CSS، يتم تحديث تصميم جميع الصفحات المرتبطة به.
  • خصائص متقدمة: توفر CSS خصائص تصميم متقدمة أكثر بكثير من خصائص HTML المدمجة.
  • بناء صفحات متجاوبة: تمكنك من جعل تصميم الصفحة يتكيف مع مختلف أحجام الشاشات.
  • تخصيص الطباعة: توفر خصائص لتحديد كيفية ظهور الصفحة عند طباعتها على الورق.

هل أنا جاهز لتعلم CSS؟

قبل البدء بهذه الدورة يجب أن تكون متقناً للغة HTML، لأن هذه اللغة تستخدم معها بشكل أساسي. إذا لم يسبق لك أن تعلمت لغة HTML، فلا تقلق أبداً.
