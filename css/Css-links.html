تصميم الروابط

الرابط في الصفحة يمكن أن يأتي بأشكال عديدة: كنص تحته خط، أيقونة صغيرة، صورة، أو حتى زر. في هذا الدرس، ستتعلم كيف تتحكم في مظهر الروابط وتأثيراتها.

خصائص الروابط

الرابط يتم وضعه بالوسم <a>. افتراضياً، يظهر كنص لونه أزرق وتحته خط. يمكن تغيير هذا المظهر باستخدام خصائص CSS.

مثال: تغيير اللون وإزالة الخط

a {
    color: red;
    text-decoration: none;
}
رابط أحمر بدون خط

حالات الروابط

الرابط يملك أربع حالات أساسية يمكنك التحكم في مظهرها عند تفاعل المستخدم معه. يجب وضع هذه الحالات بالترتيب التالي: a:link, a:visited, a:hover, a:active.

مثال 1: تحديد خصائص لكل حالة

/* رابط لم يتم فتحه بعد */
a:link { color: green; }

/* رابط تم فتحه من قبل */
a:visited { color: gray; }

/* عند تمرير الماوس فوقه */
a:hover { color: purple; }

/* عند النقر عليه */
a:active { color: red; }
رابط لتجربة الحالات

مثال 2: تحديد خصائص حالة التمرير فقط

a {
    color: green;
    text-decoration: none;
}
a:hover {
    text-decoration: underline;
}
مرر الماوس فوقي

مثال 3: تجميع حالات الروابط

يمكنك تجميع الحالات المتشابهة في نفس القاعدة. 

a:link, a:visited {
    color: green;
    text-decoration: none;
}
a:hover, a:active {
    text-decoration: underline;
}

عرض الرابط بأشكال مختلفة

يمكنك تحويل الرابط من نص إلى أي شكل آخر باستخدام خصائص CSS.

مثال 1: رابط كزر بسيط

a {
    display: inline-block;
    color: white;
    background: #1e87f0;
    ...
}
رابط كزر

مثال 2: زر يظهر عند التمرير

a {
    background: none;
    color: black;
    ...
}
a:hover {
    color: white;
    background-color: #0b5ed7;
}
زر يظهر عند التمرير

مثال 3: زر مع أيقونة

a { padding: 5px 15px; ... }
a .material-icons { font-size: 24px; ... }
downloadتحميل

مثال 4: زر يملأ السطر

a {
    display: block;
    text-align: center;
    ...
}

حل مشكلة تعليق التأثير في شاشات اللمس

كحل لمشكلة عدم رصد شاشات اللمس لتأثير :hover، يمكن وضع قواعد :hover داخل استعلام وسائط (media query) خاص.

مثال: استخدام استعلام الوسائط

@media(hover: hover) and (pointer: fine) {
    a:hover {
        background-color: #0b5ed7;
        border-color: #0a58ca;
    }
}
