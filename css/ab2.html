مفهوم تحديد العناصر

تحديد العناصر (Select Elements) هي الطرق التي يمكنك من خلالها اختيار الوسوم التي سيتم تطبيق خصائص CSS عليها. في هذا الدرس، ستتعرف على الطرق الأساسية والبسيطة التي تسمى مُحدِّدات بسيطة (Simple Selectors).

تحديد العنصر من خلال إسمه

لتطبيق خصائص على جميع الوسوم التي تملك اسماً معيناً، اذكر اسم الوسم فقط.

مثال

p {
    color: red;
    text-align: center;
}

هذا النص سيظهر باللون الأحمر وفي الوسط.

تحديد العنصر بواسطة الخاصية Class

يمكنك تحديد العناصر من خلال أسماء الكلاسات التي تمنحها لهم. لتعريف كلاس في CSS، ابدأ بنقطة (.) يتبعها اسم الكلاس.

مثال

.center {
    color: red;
    text-align: center;
}

هذا النص سيطبق عليه خصائص الكلاس.

تحديد العنصر بواسطة الخاصية ID

يمكنك تحديد عنصر واحد فريد في الصفحة بواسطة الخاصية id. لتعريف ID في CSS، ابدأ برمز المربع (#) يتبعه اسم الـ ID.

مثال

#demo {
    color: blue;
    text-align: center;
}

هذا النص سيطبق عليه خصائص الـ ID.

تحديد جميع عناصر الصفحة بواسطة *

يمكن استخدام رمز النجمة (*) لتحديد جميع العناصر في الصفحة وتطبيق الخصائص عليها.

مثال

* {
    color: blue;
    text-align: center;
}

هذا النص سيطبق عليه خصائص المحدد الشامل.

تحديد العناصر المباشرة بواسطة >

يستخدم الرمز (>) لتحديد العناصر التي هي أبناء مباشرون لعنصر محدد.

مثال

article > p {
    color: red;
}

هذا النص ابن مباشر لـ `article`.

هذا النص ليس ابناً مباشراً لـ `article`.

تجميع الخصائص المشتركة

يمكنك تحديد أكثر من عنصر في وقت واحد وتطبيق نفس الخصائص عليها، وذلك بفصل أسماء المُحدِّدات بفاصلة (,).

مثال

p, b {
    color: red;
}

هذا النص في وسم `p`.

وهذا النص في وسم `b`.

تحديد الوسم نفسه أكثر من مرة

يمكنك إعطاء وسم واحد أكثر من كلاس (class)، وسيتم تطبيق جميع الخصائص من كل الكلاسات عليه.

مثال

.red { color: red; }
.center { text-align: center; }

هذا النص يجمع بين خصائص الكلاسين.

تضارب أسماء الخصائص المستخدمة

إذا تم إعطاء العنصر قيمتين مختلفتين لنفس الخاصية، فإن القيمة الأخيرة المحددة هي التي ستُطبق.

مثال

.red { color: red; }
.blue { color: blue; }

لون هذا النص سيكون أزرق.
