مفهوم الحدود الخارجية

الحدود الخارجية (Outline) هي خطوط يمكن إظهارها حول العناصر من كل الجهات، ولكنها تختلف عن الحدود العادية (Border) في أنها لا تأخذ مساحة من حجم العنصر.

Outline apear arround border
Rounded border with rounded outline

تُستخدم الحدود الخارجية عادةً لأغراض مؤقتة مثل إظهار حدود العنصر الذي يتم التفاعل معه (مثل زر تم النقر عليه)، أو لأغراض تصحيح الأخطاء.

تحديد شكل الحدود الخارجية

لتحديد شكل الحدود الخارجية، نستخدم الخاصية outline-style مع إحدى القيم التالية: dotted، dashed، solid، double، groove، ridge، inset، outset، أو none.

مثال: أشكال الحدود المختلفة

.dotted { outline-style: dotted; }
.dashed { outline-style: dashed; }
... (وهكذا لبقية الأشكال)

تحديد سمك الحدود الخارجية

نستخدم الخاصية outline-width لتحديد سمك الحدود الخارجية، ويمكن استخدام قيم مثل: thin، medium، thick، أو رقم بالبكسل.

مثال: تحديد سمك الحدود

.one { outline-width: 2px; }
.two { outline-width: medium; }

تحديد لون الحدود الخارجية

نستخدم الخاصية outline-color لتلوين الحدود الخارجية. يجب تحديد outline-style أولاً لكي يظهر اللون.

مثال: ألوان مختلفة للحدود

.one { outline-color: red; }
.two { outline-color: green; }
Red
Green
Blue

تعريف كل خصائص الحدود الخارجية بسطر واحد

لتحديد السمك، الشكل، واللون في سطر واحد، استخدم خاصية outline.
outline: [outline-width] [outline-style] [outline-color];

مثال: استخدام الاختصار

.demo {
    outline: 5px solid red;
}
5px solid red

إبعاد الحدود الخارجية عن الحدود الأصلية

نستخدم الخاصية outline-offset لإضافة مسافة بين حدود العنصر الأصلية وحدوده الخارجية.

مثال: إزاحة الحدود

.demo {
    border: 2px solid black;
    outline: 2px solid cadetblue;
    outline-offset: 5px;
}
With Offset

مثال: مع الحشوة (Padding)

.demo {
    padding: 10px;
    border: 2px solid black;
    outline: 2px solid cadetblue;
    outline-offset: 5px;
}
With Padding

طريقة معرفة العنصر الذي يسبب مشكلة في حجم الصفحة

عند ظهور شريط تمرير غير مرغوب فيه (Scrollbar)، يمكنك استخدام الحدود الخارجية لتحديد العنصر المسبب للمشكلة. ببساطة، أضف الكود التالي لتطبيق حدود خارجية حمراء على جميع العناصر في الصفحة: 

* {
    outline: 1px solid red;
}

هذا سيجعل من السهل جداً رؤية أي عنصر يتطلب مساحة إضافية ويسبب المشكلة.
