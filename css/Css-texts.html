خصائص النصوص

توفر CSS العديد من الخصائص التي تمكنك من تحسين طريقة ظهور النصوص وتنسيقها بالشكل الذي تريده بالضبط، مثل تغيير اللون، والاتجاه، والمسافات، وغيرها من الأمور التي سنتعرف عليها في هذا الدرس.

TEXT FORMATTING

تحديد لون النص

نستخدم الخاصية color لتلوين نص العنصر. يمكن تحديد اللون بالاسم أو بنظام الألوان مثل HEX أو RGB.

المثال الأول: لون النص والعنوان

body { color: gray; }
h1 { color: cadetblue; }

عنوان باللون الأزرق

هذا النص سيأخذ لون الجسم الرئيسي (رمادي).

المثال الثاني: لون النص والخلفية

button {
    color: white;
    background-color: #0d6efd;
}

تحديد اتجاه النص

لتحديد المحاذاة الأفقية للنص، نستخدم الخاصية text-align.

المثال الأول: المحاذاة الأساسية

.left { text-align: left; }
.right { text-align: right; }
.center { text-align: center; }

هذا النص محاذى لليسار.

هذا النص محاذى لليمين.

هذا النص في الوسط.

المثال الثاني: خاصية justify

div {
    text-align: justify;
}

هذه الفقرة تستخدم خاصية justify التي تقوم بضبط المسافات بين الكلمات بحيث تكون جميع الأسطر بنفس الطول.

تحديد اتجاه الكتابة

تحديد اتجاه الكتابة أمر مهم للنصوص العربية واللغات الأخرى. نستخدم الخاصية direction.

مثال: اتجاه من اليمين إلى اليسار (RTL)

.ar {
    direction: rtl;
    text-align: right;
}

هذا النص مكتوب باللغة العربية، وجميع الأسطر تبدأ من اليمين إلى اليسار.

المسافات بين الأسطر والكلمات

يمكنك التحكم في المسافات بين الأسطر والأحرف والكلمات لتحسين قابلية القراءة.

ارتفاع السطر (Line Height)

تستخدم line-height لتحديد المسافة بين أسطر النص.

p {
    line-height: 30px;
}

هذه الفقرة لها ارتفاع سطر محدد، مما يجعل الأسطر متباعدة عن بعضها أكثر. هذا يساعد في تحسين قابلية القراءة.

توسيط النص عمودياً وأفقياً

يمكن استخدام line-height لتوسيط سطر واحد من النص عمودياً داخل عنصر ما.

.centered {
    height: 100px;
    line-height: 100px;
    text-align: center;
}
نص متوسط

مسافة بداية النص (Text Indent)

تستخدم text-indent لإضافة مسافة في بداية السطر الأول من النص.

p {
    text-indent: 50px;
}

هذا النص يبدأ بمسافة بادئة بمقدار 50 بكسل. هذه الخاصية مفيدة لتصميم الفقرات بشكل تقليدي.

مسافات الأحرف (Letter Spacing) والكلمات (Word Spacing)

تستخدم letter-spacing و word-spacing للتحكم في المسافات بين الأحرف والكلمات.

.far-chars { letter-spacing: 5px; }
.close-chars { letter-spacing: -2px; }
.far-words { word-spacing: 8px; }
.close-words { word-spacing: -2px; }

تحديد المحاذاة العمودية

نستخدم الخاصية vertical-align لتحديد المحاذاة العمودية للعناصر الموجودة ضمن سطر من النص، مثل الصور أو الأيقونات.

مثال: المحاذاة العمودية للصور

.top { vertical-align: top; }
.middle { vertical-align: middle; }
.bottom { vertical-align: bottom; }

تحويل أحرف النص

تستخدم text-transform لتحويل أحرف النص إلى حالة معينة (أحرف كبيرة، أحرف صغيرة، أو الحرف الأول من كل كلمة).

مثال: تحويل الأحرف

.uppercase { text-transform: uppercase; }
.lowercase { text-transform: lowercase; }
.capitalize { text-transform: capitalize; }
