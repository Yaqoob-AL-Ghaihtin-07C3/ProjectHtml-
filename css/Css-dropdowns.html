مفهوم القوائم المنسدلة

القائمة المنسدلة (Dropdown Menu) هي قائمة مخفية تظهر عند التفاعل مع عنصر معين في الصفحة، مثل تمرير مؤشر الماوس فوق زر أو نص، أو عند النقر عليه.

أساسيات بناء القائمة المنسدلة

لبناء قائمة منسدلة، نستخدم خاصيتي position: relative على العنصر الحاوي وخاصية position: absolute على القائمة نفسها. لجعل القائمة مخفية افتراضياً، نستخدم display: none، ثم نغيرها إلى display: block عند تمرير الماوس فوق الحاوية.

مثال: قائمة منسدلة أساسية

.dropdown { position: relative; }
.dropdown-content { display: none; position: absolute; }
.dropdown:hover .dropdown-content { display: block; }

تحديد جهة ظهور القائمة المنسدلة

يمكنك استخدام خصائص top، right، left، و bottom لتحديد مكان ظهور القائمة المنسدلة.

مثال: ظهور القائمة من الجهة اليمنى

.dropdown-content {
    right: 0;
}
Link 1 Link 2 Link 3

تحديد نوع محتوى القائمة المنسدلة

يمكن أن تحتوي القائمة المنسدلة على أي نوع من المحتوى، مثل الصور أو النصوص أو حتى قوائم منسدلة أخرى.

المثال الأول: قائمة منسدلة تحتوي على صورة

<div class="dropdown">
    <img class="dropdown-img" src="..."/>
    <div class="dropdown-content">
        <img src="..."/>
        <p>A beautiful bird.</p>
    </div>
</div>

طائر جميل.

المثال الثاني: قائمة منسدلة داخل قائمة رئيسية

<ul>
    <li><a href="#">...</a></li>
    <li class="dropdown">
        <a href="javascript:void(0)">Submenu</a>
        <div class="dropdown-content">
            ...
        </div>
    </li>
</ul>

المثال الثالث: قوائم منسدلة متداخلة (Nested Menus)

هذا النوع من القوائم يسمح بإنشاء قوائم فرعية داخل القوائم الفرعية نفسها. 

nav ul ul {
    display: none;
    position: absolute;
    top: 45px;
}
nav ul li:hover > ul {
    display: inherit;
}
