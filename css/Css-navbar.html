مفهوم قوائم الروابط

المبادئ الأساسية لتصميم القوائم

عادةً ما يتم تصميم قائمة الروابط باستخدام الوسمين <ul> و <li>، مع وضع الرابط داخل وسم <a>. بعد ذلك، يتم إزالة النقاط والهوامش الافتراضية باستخدام CSS.

مثال: إزالة النقاط والهوامش

ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
}

تصميم قائمة عمودية

في القائمة العمودية، يظهر كل رابط على سطر منفرد ويأخذ العرض الكامل للقائمة.

المثال الأول: تصميم أساسي

ul {
    width: 220px;
    background-color: #f1f1f1;
}
li a {
    display: block;
    padding: 12px 16px;
}
li a:hover {
    background-color: #444;
    color: white;
}

تصميم قائمة أفقية

تظهر الروابط في القائمة الأفقية على نفس السطر.

المثال الأول: تصميم أساسي

ul {
    background-color: #282a35;
    overflow: hidden;
}
li {
    float: left;
}
li a {
    display: block;
    padding: 12px 16px;
    color: white;
}
li a:hover {
    background-color: black;

تحديد موقع القائمة

يمكنك تثبيت القائمة في مكان معين من الصفحة باستخدام خاصية position.

المثال الأول: قائمة ثابتة في الأعلى

ul {
    position: fixed;
    top: 0;
    width: 100%;
}

ملاحظة: هذه القائمة ستظل مرئية في أعلى الصفحة حتى عند التمرير للأسفل.

المثال الثاني: قائمة ثابتة في الأسفل

ul {
    position: fixed;
    bottom: 0;
    width: 100%;
}

ملاحظة: هذه القائمة ستظل مرئية في أسفل الصفحة حتى عند التمرير.

المثال الثالث: قائمة تثبت عند التمرير (Sticky)

ul {
    position: sticky;
    top: 0;
    width: 100%;
}

ملاحظة: هذه القائمة ستتبع التمرير حتى تصل إلى الأعلى ثم تثبت في مكانها.

المثال الرابع: قائمة ثابتة على اليسار

ul {
    position: fixed;
    left: 0;
    width: 25%;
    height: 100%;
}

ملاحظة: هذه القائمة ستبقى ثابتة في الجهة اليسرى من النافذة.
