مفهوم قوائم التعداد

قوائم التعداد نوعان:

  • قوائم مرتبة (Ordered Lists): يبدأ فيها كل عنصر برقم يدل على موقعه.
  • قوائم غير مرتبة (Unordered Lists): يبدأ فيها كل عنصر برمز (مثل دائرة أو مربع) أو صورة.
الرقم أو الرمز الذي يظهر في بداية العنصر يسمى مُحدّد العنصر (Item marker).

تحديد نوع الترقيم والتنقيط

يمكنك تحديد نوع المحددات باستخدام الخاصية list-style-type.

المثال الأول: أنواع الترقيم (للقائمة المرتبة)

.arabic-indic { list-style-type: arabic-indic; }
.decimal-leading-zero { list-style-type: decimal-leading-zero; }
  1. لغة البرمجة
  2. لغة الترميز
  3. لغة التنسيق
  1. Decimal (1, 2, 3)
  2. Decimal
  3. Decimal
  1. Decimal (01, 02, 03)
  2. Decimal
  3. Decimal
  1. Lower Alpha (a, b, c)
  2. Lower Alpha
  3. Lower Alpha

.disc { list-style-type: disc; }

عرض صورة في بداية كل عنصر

يمكنك استخدام الخاصية list-style-image لعرض صورة أو تدرج لوني كـ مُحدّد للعنصر.

المثال الأول: عرض صورة

ul {
    list-style-image: url('circle.png');
}

تحديد مكان محددات العناصر

خاصية list-style-position تتحكم في مكان ظهور المحددات، إما داخل أو خارج حدود العنصر.

مثال: inside vs outside

ul .inside { list-style-position: inside; }
ul .outside { list-style-position: outside; }

تعديل تصميم القائمة والعناصر

يمكنك تخصيص تصميم القوائم والعناصر والمحددات باستخدام محددات CSS.

مثال: تعديل شامل للتصميم

ul {
    margin: 0;
    background: deepskyblue;
    ...
}
ul li {
    background: darkslateblue;
    color: white;
    ...
}
ul li::marker {
    color: gold;
}

عرض أيقونة في بداية العنصر

يمكن استخدام أيقونات من مكتبات مثل Font Awesome كمحددات للقوائم.

مثال: استخدام أيقونة Font Awesome

ul { list-style-type: '\f006'; }
ul li::marker { font-family: "Font Awesome 6 Free"; font-weight: 900; }
