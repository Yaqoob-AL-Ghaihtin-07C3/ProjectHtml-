طرق تحديد الألوان

عدد الألوان التي يمكنك اختيارها في عالم الويب هو بالملايين. لست مضطراً لحفظ أسماء الألوان أو درجاتها، حيث يمكنك استخدام أدوات جاهزة تعطيك كود اللون الذي تريده.

بشكل عام، الألوان الرئيسية هي الأحمر (Red) والأخضر (Green) والأزرق (Blue)، ومن هنا جاء اسم **RGB**.

هناك 6 طرق يمكنك اتباعها لتحديد الألوان في CSS:

  • كتابة اسم اللون (Color Name).
  • تحديد درجة اللون بأسلوب HEX.
  • تحديد درجة اللون بأسلوب RGB.
  • تحديد درجة اللون وشفافيته بأسلوب RGBA.
  • تحديد درجة اللون والتشبّع والإضاءة بأسلوب HSL.
  • تحديد درجة اللون والتشبّع والإضاءة والشفافية بأسلوب HSLA.

التلوين بأسلوب Color Names

هذا الأسلوب يكون بكتابة اسم اللون فقط، مع العلم أن CSS فيها 140 لونًا جاهزًا يمكنك استخدامهم.

أسماء الألوان يمكنك كتابتها بأحرف صغيرة أو كبيرة دون أي مشكلة (مثال: red, Red, RED).

في المثال التالي قمنا بتلوين نصوص الفقرات بأسماء ألوان مختلفة. 

<p style="color:Tomato;">Tomato color</p>
<p style="color:DodgerBlue;">DodgerBlue color</p>
<p style="color:MediumSeaGreen;">MediumSeaGreen color</p>

التلوين بأسلوب HEX

هذا الأسلوب يكون بوضع رمز المربع # يليه 3 أو 6 أرقام وأحرف تمثل اللون. الأرقام بين 0-9 والأحرف بين A-F.

الفرق بين استخدام 3 أو 6 أحرف

  • 3 أحرف: الأول يمثل درجة الأحمر، الثاني الأخضر، والثالث الأزرق.
  • 6 أحرف: كل حرفين يمثلان درجة لون (الأول والثاني للأحمر، الثالث والرابع للأخضر، الخامس والسادس للأزرق).

في المثال التالي قمنا بتلوين نصوص الفقرات بألوان مختلفة بأسلوب HEX. 

p {
    color: #FF6347;
}
p {
    color: #1E90FF;
}
p {
    color: #3CB371;
}

التلوين بأسلوب RGB

هذا الأسلوب يكون بتمرير ثلاثة أعداد بين 0-255 للدالة rgb()، تمثل درجات الأحمر، الأخضر، والأزرق على التوالي. 

rgb(Red, Green, Blue)

في المثال التالي قمنا بتلوين نصوص الفقرات بألوان مختلفة بأسلوب RGB. 

p {
    color: rgb(255, 99, 71);
}
p {
    color: rgb(30, 144, 255);
}
p {
    color: rgb(60, 179, 113);
}

التلوين بأسلوب RGBA

هذا الأسلوب هو نفسه أسلوب RGB، ولكنه يضيف خاصية الشفافية (Alpha) بقيمة بين 0-1. 

rgb(Red, Green, Blue, Alpha)

في المثال التالي قمنا بتلوين نصوص الفقرات بلون واحد مع درجات شفافية مختلفة. 

p {
    color: rgba(255, 0, 0, 1);
}
p {
    color: rgba(255, 0, 0, 0.8);
}
p {
    color: rgba(255, 0, 0, 0.6);
}

التلوين بأسلوب HSL

يعتمد هذا الأسلوب على تحديد اللون الأساسي (Hue)، نسبة التشبّع (Saturation)، ونسبة الإضاءة (Lightness). 

hsl(Hue, Saturation, Lightness)
  • Hue: قيمة بين 0-360 تمثل اللون الأساسي.
  • Saturation: قيمة بين 0-100% تمثل درجة التشبّع.
  • Lightness: قيمة بين 0-100% تمثل درجة الإضاءة.

في المثال التالي قمنا بتلوين نصوص الفقرات بألوان مختلفة بأسلوب HSL. 

p {
    color: hsl(248, 53%, 58%);
}
p {
    color: hsl(147, 50%, 47%);
}
p {
    color: hsl(39, 100%, 50%);
}

التلوين بأسلوب HSLA

هذا الأسلوب هو نفسه أسلوب HSL، ولكن مضافًا إليه خاصية الشفافية (Alpha) بقيمة بين 0-1. 

hsla(Hue, Saturation, Lightness, Alpha)

في المثال التالي قمنا بتلوين نصوص الفقرات بلون واحد مع درجات شفافية مختلفة. 

p {
    color: hsla(240, 100%, 50%, 1);
}
p {
    color: hsla(240, 100%, 50%, 0.8);
}
p {
    color: hsla(240, 100%, 50%, 0.6);
}
