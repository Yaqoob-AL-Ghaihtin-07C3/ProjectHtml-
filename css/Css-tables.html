خصائص الجداول

باستخدام خصائص CSS، يمكنك تنسيق الجداول بشكل احترافي، من ضبط شكل الحدود وتلوين الأسطر إلى جعلها متجاوبة (Responsive) لتظهر بشكل مناسب على جميع أحجام الشاشات.

تحسين تصميم حدود الجدول

افتراضياً، لا تظهر الجداول أي حدود، ولكن يمكنك إظهارها وتنسيقها باستخدام خاصية border.

المثال الأول: حدود خارجية للجدول

table {
    border: 1px solid;
}

دمج حدود الجدول

لحل مشكلة الحدود المزدوجة، استخدم خاصية border-collapse: collapse; على الجدول. 

table {
    border-collapse: collapse;
}
table, th, td {
    border: 1px solid;
}

تحديد حجم الجدول

يمكنك تحديد حجم الجدول وعرض أعمدته باستخدام خاصية width.

المثال الأول: عرض الجدول بالكامل

table {
    width: 100%;
}
عرض 100%

المثال الثاني: تحديد حجم الأعمدة

<table style="width:100%">
    ...
    <th style="width:70%">User</th>
    <th style="width:30%">Country</th>
    ...
</table>

تحديد اتجاه ومحاذاة محتوى الجدول

يمكنك التحكم في المحاذاة الأفقية (text-align) والعمودية (vertical-align) لمحتوى الخلايا.

مثال: محاذاة المحتوى

th, td {
    text-align: left;
    vertical-align: top;
}

تصميم جدول متجاوب

لجعل الجدول متجاوباً، قم بوضعه داخل <div> مع الخاصية overflow-x: auto;. هذا سيضيف شريط تمرير أفقي إذا كان الجدول أكبر من الشاشة.

مثال: جدول متجاوب

<div style="overflow-x:auto;">
    <table>...</table>
</div>

تحديد مكان ظهور وصف الجدول

يمكنك تغيير موقع وصف الجدول <caption> باستخدام خاصية caption-side.

مثال: وصف الجدول في الأسفل

table { caption-side: bottom; }
caption { margin-top: 7px; }
