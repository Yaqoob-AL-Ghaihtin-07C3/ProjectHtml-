مفهوم تنسيق الصور والفيديوهات

إذا كانت المساحة المخصصة لعرض الصور أو الفيديوهات غير مساوية لحجمها، يمكنك تحديد كيف سيتم عرضها فيها. أي يمكنك تحديد ما إن كان يجب تصغيرها، تكبيرها أو تمديدها حتى تملأ كل المساحة المتاحة.

كيفية ضبط عرض الصورة أو الفيديو

لتحديد كيفية عرض الصورة أو الفيديو، استخدم الخاصية object-fit مع إحدى القيم التالية:

  • fill: القيمة الافتراضية، تجعل العنصر يملأ المساحة المتاحة بغض النظر عن نسبة أبعاده.
  • contain: يغير حجم العنصر للحفاظ على نسبة أبعاده مع ضمان ظهوره بالكامل داخل المساحة.
  • cover: يغير حجم العنصر ليمتلئ به المساحة بالكامل مع الحفاظ على نسبة أبعاده، مما قد يؤدي إلى اقتصاص جزء من العنصر.
  • none: لا يغير حجم العنصر.
  • scale-down: يطبق إما none أو contain، حسب أيهما يجعل العنصر أصغر.

مثال

img {
    width: 200px;
    height: 150px;
}
.fill { object-fit: fill; }
.contain { object-fit: contain; }
.cover { object-fit: cover; }
.none { object-fit: none; }
.scale-down { object-fit: scale-down; }

كيفية تحديد موقع الصورة أو الفيديو

تُستخدم الخاصية object-position لتحديد الإحداثيات التي سيتم عرض الصورة أو الفيديو عليها داخل المساحة المخصصة، خاصة عندما يكون حجمها أكبر من المساحة المتاحة.

مثال

.position-example-1 { object-position: 20px 20px; }
.position-example-2 { object-position: 30% top; }
.position-example-3 { object-position: center top; }
.position-example-4 { object-position: center center; }
.position-example-5 { object-position: center bottom; }
