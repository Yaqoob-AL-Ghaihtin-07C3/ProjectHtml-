مفهوم التحويلات الثلاثية الأبعاد

توفر لك CSS مجموعة من الدوال لإجراء تحويلات ثلاثية الأبعاد (3D Transforms) على العناصر، مثل إزاحتها أو تدويرها على ثلاث محاور: X، Y، و Z.

المحاور الثلاثية الأبعاد X و Y و Z

في عالم ثلاثي الأبعاد، نتعامل مع ثلاث محاور متقاطعة في نقطة تسمى نقطة التحوّل (Transform Origin). هذه النقطة هي الأساس لأي تحويلات، وتكون افتراضياً في وسط العنصر.

يُعتبر المحور Z هو المحور الذي يحدد العمق، حيث كلما اقتربت من العنصر على هذا المحور بدا أكبر، وكلما ابتعدت عنه بدا أصغر.

مفهوم المنظور الثلاثي الأبعاد

لإظهار العنصر من منظور ثلاثي الأبعاد، يجب تحديد المسافة بين نقطة التحوّل والمحور Z. يتم ذلك باستخدام الخاصية perspective على العنصر الحاوي (الأب).

تدوير العنصر الثلاثي الأبعاد

للتدوير، يمكنك استخدام دوال مثل rotateX()، rotateY()، rotateZ()، أو rotate3d().

المثال الأول (تدوير على المحور X)

#demo {
    transform: rotateX(20deg);
}
تدوير X

تحريك العنصر الثلاثي الأبعاد

لتحريك العنصر، استخدم دوال مثل translateX()، translateY()، translateZ()، أو translate3d().

المثال الثالث (تحريك على المحور Z)

لاحظ أن تحريك العنصر على المحور Z يجعله يبدو أكبر أو أصغر.

#demo {
    transform: translateZ(40px);
}
تحريك Z

تحجيم العنصر الثلاثي الأبعاد

لتكبير أو تصغير العنصر، استخدم دوال مثل scaleX()، scaleY()، scaleZ()، أو scale3d().

المثال الثالث (تحجيم على المحور Z)

في العناصر ثنائية الأبعاد، لن تلاحظ أي تغيير. تأثير هذه الخاصية يظهر مع المجسمات ثلاثية الأبعاد.

#demo {
    transform: scaleZ(1.5);
}
تحجيم Z

إمالة العنصر الثلاثي الأبعاد

لإمالة العنصر، استخدم دوال مثل skewX() أو skewY().

المثال الأول (إمالة 20 درجة على المحور X)

#demo {
    transform: skewX(20deg);
}
إمالة X

تحديد موقع نقطة التحول

يمكنك استخدام الخاصية transform-origin لتغيير موقع نقطة التحول التي تتم حولها التحويلات.

مثال (تغيير نقطة التحول إلى "left top")

#demo {
    transform-origin: left top;
    transform: rotateZ(20deg);
}
نقطة تحول

تصميم عنصر ثلاثي الأبعاد

فيما يلي مثال على بناء مكعّب ثلاثي الأبعاد يتألف من 6 أوجه، باستخدام خاصية perspective على العنصر الأب و transform-style: preserve-3d. 

.cube {
    perspective: 500px;
    transform-style: preserve-3d;
}

إظهار العنصر بشكل مسطح أو ثلاثي الأبعاد

تُستخدم الخاصية transform-style على العنصر الأب لتحديد ما إذا كان يجب معاملة العناصر الأبناء كأجسام ثلاثية الأبعاد أم كأجسام مسطحة.

مثال (مع preserve-3d)

.cube {
    transform-style: preserve-3d;
}

تغيير موقع نقطة منظور العنصر الثلاثي الأبعاد

يمكنك استخدام الخاصية perspective-origin على العنصر الأب لتحديد من أي نقطة يتم عرض المنظور.

مثال

.cube-container {
    perspective-origin: 150% 100%;
}
