ما هو Grid

Grid هو مجموعة من الخصائص التي يمكنك استخدامها لتحديد مواقع وأحجام العناصر بشكل مرن جداً، مما يجعله مثالياً لتنظيم مخططات التصميم المعقدة ثنائية الأبعاد.

البدء باستخدام Grid

للبدء باستخدام Grid، يجب وضع العناصر بداخل حاوية (container) وإعطاء هذه الحاوية الخاصية display: grid;. 

.grid-container {
    display: grid;
}

بعد هذه الخطوة، يمكنك استخدام باقي خصائص Grid لترتيب العناصر بالشكل الذي تريده.

الفرق بين Flex و Grid

Flexbox: يعتبر خياراً مثالياً لترتيب العناصر في بعد واحد (إما صف أو عمود).

Grid: هو الخيار الأفضل لتنظيم العناصر في بعدين (صفوف وأعمدة) في نفس الوقت، مما يتيح لك بناء مخططات معقدة.

يمكنك استخدام Flexbox و Grid معاً في نفس المشروع للحصول على أفضل النتائج.

الخاصية grid-auto-flow

تحدد كيفية وضع العناصر غير المحددة في الشبكة تلقائياً.

القيمة الاستخدام
row يرتب العناصر في صفوف (افتراضي).
column يرتب العناصر في أعمدة.
dense يرتب العناصر في المساحات الفارغة لتجنب الفجوات.

مثال (row)

.grid-container {
    grid-auto-flow: row;
}
.item-1, .item-2 {
    grid-column: auto/span 2;
}

الخاصية grid-template-areas

تسمح لك بتسمية مناطق في الشبكة ووضع العناصر فيها بالاسم.

مثال

.grid-container {
    display: grid;
    grid-template-areas:
        "header header"
        "navbar main"
        "navbar footer";
}
.header { grid-area: header; }
.navbar { grid-area: navbar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

الخاصية grid-template (اختصار)

تجمع الخصائص grid-template-areas, grid-template-rows, و grid-template-columns في خاصية واحدة.

مثال

.grid-container {
    display: grid;
    grid-gap: 10px;
    grid-template:
        "header header" 90px
        "navbar main" 1fr
        "navbar footer" 50px
        / 150px auto;
}
