ما هي دوال الحسابات؟

توفر CSS دوال برمجية (Functions) يمكن استخدامها لإجراء عمليات حسابية لتعيين أحجام العناصر بشكل مرن وبسيط جداً، مما يساعد كثيراً عند بناء تصاميم متجاوبة (Responsive) تظهر بشكل مناسب على مختلف أحجام الشاشات.

الدالة calc()

تُستخدم هذه الدالة لإجراء عمليات حسابية مثل الجمع (+)، الطرح (-)، الضرب (*)، والقسمة (/) عند تحديد القياسات.

مثال: استخدام calc() لتحديد العرض

#calc-demo {
    position: absolute;
    left: 40px;
    width: calc(100% - 80px);
}
عرض: 100% - 80px

هذا الكود يجعل عرض العنصر يساوي كامل عرض الصفحة (100%) مطروحًا منه 80 بكسل، مما يمنحه مساحة 40 بكسل على اليمين و 40 بكسل على اليسار.

الدالة max()

تُستخدم هذه الدالة لوضع عدة قيم، ويقوم المتصفح باختيار القيمة الأكبر منها.

مثال: استخدام max()

#max-demo {
    width: max(50%, 300px);
}
العرض: max(50%, 300px)

في هذا المثال، سيأخذ العنصر العرض الأكبر بين 50% من عرض الصفحة و 300 بكسل.

الدالة min()

تُستخدم هذه الدالة لوضع عدة قيم، ويقوم المتصفح باختيار القيمة الأصغر منها.

مثال: استخدام min()

#min-demo {
    width: min(50%, 300px);
}
العرض: min(50%, 300px)

في هذا المثال، سيأخذ العنصر العرض الأصغر بين 50% من عرض الصفحة و 300 بكسل.
