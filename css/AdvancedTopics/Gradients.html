مفهوم تدرج الألوان

تدرّج الألوان (Gradient) هو عرض عدة ألوان بجانب بعضها البعض وبشكل انسيابي بدلاً من استخدام لون واحد فقط.

Linear Gradient

توّفر لك CSS مجموعة من الدوال الجاهزة التي يمكنك استخدامها لإنشاء تدرجات بأشكال مختلفة: مستقيمة، دائرية، أو أسطوانية، مع إمكانية التحكم في اتجاهها وتكرارها.

أساسيات تدرّج الألوان

يمكن تطبيق تدرج الألوان على خلفية العنصر باستخدام الخاصية background-image أو background. يجب دائماً وضع لون خلفية احتياطي قبل اللون المتدرج في حالة عدم دعم المتصفح له.

مثال

.gradient {
    background: #00188F; /* لون احتياطي */
    background: linear-gradient(#00188F, #EC008C);
}

دوال تدرج الألوان

فيما يلي جميع الدوال التي يمكنك استخدامها للحصول على ألوان متدرجة:

1. linear-gradient()

لعرض ألوان متدرجة مستقيمة ليس فيها انحناء مع جعلها تتمدد لتملأ خلفية العنصر.

2. repeating-linear-gradient()

لعرض ألوان متدرجة مستقيمة مع جعلها تتكرر لتملأ خلفية العنصر.

3. radial-gradient()

لعرض ألوان متدرجة دائرياً مع جعلها تتمدد لتملأ خلفية العنصر.

4. repeating-radial-gradient()

لعرض ألوان متدرجة دائرياً مع جعلها تتكرر لتملأ خلفية العنصر.

5. conic-gradient()

لعرض ألوان متدرجة بشكل أسطواني مع جعلها تتمدد لتملأ خلفية العنصر.

6. repeating-conic-gradient()

لعرض ألوان متدرجة بشكل أسطواني مع جعلها تتكرر لتملأ خلفية العنصر.
