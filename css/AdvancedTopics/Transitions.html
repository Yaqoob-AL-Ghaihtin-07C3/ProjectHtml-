مفهوم التأثيرات الإنتقالية

التأثيرات الإنتقالية (Transitions) تسمح بتغيير قيم خصائص العنصر بشكل تدريجي وسلس بدلاً من التغيير المفاجئ.

مرّر مؤشر الماوس على العناصر التالية ولاحظ كيف تتغير حالتها بشكل سلس:

أساسيات التأثيرات الإنتقالية

لإنشاء تأثير انتقالي، يجب عليك تحديد الخاصية التي سيتم تغييرها و المدة التي سيستغرقها هذا التغيير.

مثال

.btn {
    width: 200px;
    transition-property: width;
    transition-duration: 1s;
}
.btn:hover {
    width: 300px;
}

تغيير قيمة عدة خصائص أثناء الإنتقال

يمكنك تطبيق التأثير الانتقالي على أكثر من خاصية عن طريق كتابة أسمائها مفصولة بفاصلة، أو باستخدام الكلمة all.

المثال الأول (عدة خصائص)

.btn {
    transition-property: width, background-color;
    transition-duration: 1s;
}

المثال الثاني (استخدام all)

.btn {
    transition-property: all;
    transition-duration: 1s;
}

تأخير التأثير الإنتقالي

باستخدام الخاصية transition-delay، يمكنك تأخير بدء التأثير لمدة محددة.

مثال

.btn {
    transition-delay: .5s;
}

تحديد منحنى سرعة الإنتقال

تُستخدم الخاصية transition-timing-function للتحكم في سرعة التأثير خلال مدته.

مثال

مرر مؤشر الماوس على الإطار أدناه لملاحظة الفرق بين منحنيات السرعة المختلفة:

linear
ease
ease-in
ease-out
ease-in-out

إضافة التأثيرات الإنتقالية بشكل مختصر

يمكنك دمج جميع خصائص التأثيرات في خاصية واحدة وهي transition.

مثال

input {
    transition: width ease-in-out .3s;
}
input:focus {
    width: 250px;
}
