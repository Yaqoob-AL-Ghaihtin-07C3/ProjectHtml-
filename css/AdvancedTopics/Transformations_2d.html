مفهوم التحويلات الثنائية الأبعاد

توفر لك CSS مجموعة من الدوال التي يمكنك استخدامها لإجراء تحويلات ثنائية الأبعاد (2D Transforms) على العناصر مثل إزاحتها من مكانها الأساسي، تدويرها، أو تغيير حجمها.

Rotate
Scale
Skew
Translate

يتم إجراء التحويلات باستخدام الخاصية transform مع تمرير إحدى دوال التحويل لها كقيمة.

تدوير العنصر

للتدوير، نستخدم الدالة rotate() مع تمرير درجة الدوران.

المثال الأول (تدوير 20 درجة)

div {
    transform: rotate(20deg);
}
نص
دوران

المثال الثاني (تدوير -20 درجة)

div {
    transform: rotate(-20deg);
}
نص
دوران

تحريك العنصر

لتحريك العنصر من مكانه، نستخدم الدالة translate().

المثال الأول (تحريك 40px أفقياً و 80px عمودياً)

div {
    transform: translate(40px, 80px);
}

تحريك

تحجيم العنصر

لتحجيم العنصر، نستخدم الدالة scale() أو scaleX() أو scaleY().

المثال الأول (تكبير 1.2)

div {
    transform: scale(1.2);
}
نص
تحجيم

إمالة العنصر

لإمالة العنصر، نستخدم الدالة skew() أو skewX() أو skewY().

المثال الأول (إمالة 20 درجة أفقياً)

div {
    transform: skewX(20deg);
}

إمالة

المثال الثاني (إمالة -20 درجة أفقياً)

div {
    transform: skewX(-20deg);
}

إمالة

تحديد كل الخصائص الثنائية الأبعاد للعنصر

لإجراء أكثر من تعديل في نفس الوقت، يمكنك استخدام الدالة matrix().

مثال

div {
    transform: matrix(1, 0, -0.50, 1, 40, 80);
}
