ما هي إستعلامات الوسائط؟

إستعلامات الوسائط (Media Queries) هي أداة قوية في CSS تسمح لك بتطبيق خصائص تصميم مختلفة بناءً على خصائص الجهاز الذي يتم عرض الصفحة عليه، مثل حجم الشاشة أو اتجاهها (طولي أو عرضي).

تُعد إستعلامات الوسائط ضرورية لبناء تصاميم متجاوبة (Responsive) تظهر بشكل مثالي على الهواتف، الأجهزة اللوحية، والحواسيب.

قواعد إستعلامات الوسائط

للبدء باستخدام إستعلامات الوسائط، نستخدم الكلمة المفتاحية @media ثم نضع الشروط التي نريدها. 

@media not|only mediatypes and (expressions) {
    /* كود CSS هنا */
}
  • @media: الكلمة المفتاحية التي تبدأ بها قاعدة الإستعلام.
  • mediatypes: نوع الوسائط (مثل screen أو print).
  • expressions: الشروط التي يجب أن تتحقق (مثل max-width: 400px).

أنواع إستعلامات الوسائط

تُحدد أنواع الوسائط (Media Types) نوع الجهاز الذي يتم عليه عرض الصفحة. إليك أهم الأنواع:

النوع الاستخدام
all لتطبيق الخصائص على جميع أنواع الوسائط.
screen لتطبيق الخصائص فقط عند عرض الصفحة على شاشة (حاسوب، هاتف، إلخ).
print لتطبيق الخصائص فقط عند طباعة الصفحة.

المثال الأول: max-width

@media screen and (max-width: 400px) {
    p {
        color: red;
    }
}

سيصبح لون هذا النص أحمر إذا كان عرض الشاشة 400 بكسل أو أقل.

المثال الثاني: ترتيب القواعد

p {
    color: gray;
}
@media screen and (max-width: 400px) {
    p {
        color: red;
    }
}

اللون الأساسي رمادي، ولكن يتغير للأحمر عند الشاشة الصغيرة.

ملاحظة: يجب وضع قواعد @media بعد القواعد الأساسية لضمان تطبيقها بشكل صحيح.

المثال الرابع: print

@media print {
    p {
        color: red;
    }
}

لون هذا النص رمادي، ولكنه سيتحول إلى أحمر عند طباعة الصفحة.

المثال الخامس والسادس: تجميع الأنواع

@media screen, print {
    p {
        color: brown;
    }
}
@media screen or print {
    p {
        color: brown;
    }
}

هذا النص لونه بني سواء على الشاشة أو عند الطباعة.

المثال السابع: not

@media not print {
    p {
        color: brown;
    }
}

هذا النص بني في جميع الحالات ما عدا عند الطباعة.

شروط إستعلامات الوسائط

يمكنك استخدام عدة شروط لتحديد متى يجب تطبيق الأنماط.

المثال الأول: min-width

@media (min-width: 400px) {
    body {
        padding: 20px;
    }
}

ملاحظة: سيتم إضافة هامش داخلي 20 بكسل للصفحة إذا كان عرض الشاشة 400 بكسل أو أكثر.

المثال الثاني: and

@media (min-width: 400px) and (max-width: 1000px) {
    body {
        padding: 20px;
    }
}

ملاحظة: سيتم تطبيق الهامش فقط إذا كان عرض الشاشة بين 400 و 1000 بكسل.

المثال الثالث: تجميع الشروط (النسخة الحديثة)

@media (400px <= width <= 1000px) {
    body {
        padding: 20px;
    }
}

هذه الطريقة الأحدث لكتابة شروط النطاق.

المثال الرابع: ترتيب الشروط منطقياً

h1 { font-size: 1.8rem; }
@media (min-width: 768px) { h1 { font-size: 2.2rem; } }
@media (min-width: 960px) { h1 { font-size: 2.6rem; } }

هذا النص يتغير حجمه حسب عرض الشاشة.

هذا مثال على كيفية استخدام @media لإنشاء تصميم متجاوب يتغير حجم الخط فيه بشكل تدريجي.

المثال الخامس: orientation

@media (orientation: landscape) {
    #orientation::before {
        content: 'Landscape Mode';
    }
}

اتجاه الشاشة:

ملاحظة: قم بتغيير اتجاه شاشتك (طولي/عرضي) لمشاهدة تأثير هذا الكود.

المثال السادس: or

@media (min-height: 680px), screen and (orientation: portrait) {
    body {
        padding: 20px;
    }
}

سيتم تطبيق الهامش إذا كان طول النافذة 680 بكسل على الأقل أو إذا كانت الشاشة في الوضع العمودي.

هذا النص جزء من الصفحة التي سيتغير الهامش فيها.
