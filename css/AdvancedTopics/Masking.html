مفهوم التقنيع

التقنيع (Masking) هو أسلوب يستخدم لإظهار أجزاء معينة من عنصر (مثل صورة) أو التلاعب بشفافيته، وذلك باستخدام صورة أو لون متدرج كقناع.

كمثال بسيط، لاحظ أنه بعد تركيب القناع على الصورة الأساسية إختفت الأجزاء غير المشتركة بينهما لتظهر كما يلي.

الصورة الأصلية | القناع | الصورة بعد التقنيع

طريقة وضع قناع للعنصر

للتأكد من دعم المتصفحات، يتم استخدام البادئة -webkit- بجانب الخصائص الأساسية. 

-webkit-mask-image: url(mask.png);
mask-image: url(mask.png);

-webkit-mask-size: 100%;
mask-size: 100%;

وضع صورة كقناع

لتحديد الصورة التي سيتم استخدامها كقناع، نستخدم الخاصية mask-image ونمرر لها مسار الصورة عبر الدالة url().

مثال

img {
    -webkit-mask-image: url(mask.png);
    mask-image: url(mask.png);
}

وضع لون متدرج كقناع

باستخدام الخاصية mask-image مع دوال تدرج الألوان، يمكن إنشاء أقنعة شفافة.

المثال الأول (تدرج خطي)

img {
    -webkit-mask-image: linear-gradient(black, transparent);
    mask-image: linear-gradient (black, transparent);
}

المثال الثاني (تدرج دائري)

img {
    -webkit-mask-image: radial-gradient(circle, black 50%, rgba(0, 0, 0, 0.5) 50%);
    mask-image: radial-gradient(circle, black 50%, rgba(0, 0, 0, 0.5) 50%);
}

تحديد حجم القناع

الخاصية mask-size تسمح لك بالتحكم في حجم القناع بالنسبة للعنصر.

المثال الأول (حجم 100%)

img {
    mask-size: 100%;
}

المثال الثاني (حجم 20%)

img {
    mask-size: 20%;
}

تحديد كيف سيتم تكرار القناع

الخاصية mask-repeat تحدد كيفية تكرار القناع لملء مساحة العنصر.

المثال الأول (تكرار عمودي)

mask-repeat: repeat-y;

المثال الثاني (تكرار مع تباعد)

mask-repeat: space;

تحديد مكان القناع بالنسبة للعنصر

الخاصية mask-position تسمح لك بوضع القناع في أي مكان داخل العنصر.

مثال

mask-position: center center;

رسم قناع بواسطة SVG

يمكنك استخدام وسم <mask> داخل <svg> لرسم قناع مخصص.

المثال الأول (قناع مضلع)

<svg width="100%" height="auto">
    <mask id="svgmask">
        <polygon fill="#FFFFFF" points="50 0, 83 12, 100 43, 94 78, 68 100, 32 100, 6 78, 0 43, 17 12"></polygon>
    </mask>
    <image xlink:href="/tutorials/css/masking/nature.png" mask="url(#svgmask)"></image>
</svg>

المثال الثاني (قناع دوائر)

<svg width="100%" height="auto">
    <mask id="svgmask">
        <circle fill="#ffffff" cx="75" cy="75" r="60"></circle>
        <circle fill="#ffffff" cx="260" cy="100" r="85"></circle>
        ...
    </mask>
    <image xlink:href="/tutorials/css/masking/nature.png" mask="url(#svgmask)"></image>
</svg>
