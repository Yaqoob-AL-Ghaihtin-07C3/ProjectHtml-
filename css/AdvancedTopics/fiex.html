ما هو Flexbox

Flexbox هو مجموعة من الخصائص التي يمكنك استخدامها لترتيب العناصر بشكل مرن جداً. فهو يتيح لك التحكم في مواقع العناصر، أحجامها، وكيفية تمددها أفقياً وعمودياً، بالإضافة إلى الهوامش بينها.

البدء باستخدام Flexbox

للبدء باستخدام Flexbox، يجب وضع العناصر التي تريد ترتيبها داخل عنصر آخر (يُسمى الحاوية)، ثم إعطاء الحاوية الخاصية display: flex;. 

.flex-container {
    display: flex;
}

الخاصية flex-direction

تستخدم لتحديد اتجاه صف العناصر، أفقياً أو عمودياً.

القيمة row (افتراضي)

    flex-direction: row;

الخاصية flex-shrink

تستخدم لتقليص حجم العنصر نسبةً لباقي العناصر. القيمة الافتراضية هي 1، وكلما زادت القيمة، زاد التقلص.

مثال

.flex-container {
    flex-direction: row;
}
.item-1 { flex-shrink: 1; }
.item-2 { flex-shrink: 2; }
.item-3 { flex-shrink: 3; }

الخاصية flex-grow

تستخدم لتوسيع حجم العنصر نسبةً لباقي العناصر في المساحة المتاحة. القيمة الافتراضية هي 0.

مثال

.item-1 {
    flex-grow: 1;
}

3

الخاصية flex-basis

تستخدم لتحديد الحجم الأساسي للعنصر قبل التوزيع.

مثال: قيمة ثابتة

.item-1 {
    flex-basis: 150px;
}

الخاصية order

تستخدم لتغيير ترتيب العناصر داخل الحاوية. القيمة الافتراضية هي 0.

مثال

.item-1 { order: 4; }
.item-2 { order: 2; }

الخاصية flex (اختصار)

تُعدّ هذه الخاصية اختصاراً لـ flex-grow، flex-shrink، و flex-basis بالترتيب. 

flex: [flex-grow] [flex-shrink] [flex-basis];

مثال

.item-2 {
    flex: 0 1 300px;
}

2
3

الخاصية flex-flow (اختصار)

تُعدّ هذه الخاصية اختصاراً لـ flex-direction و flex-wrap بالترتيب. 

flex-flow: [flex-direction] [flex-wrap];

مثال

.flex-container {
    flex-flow: row wrap;
}
