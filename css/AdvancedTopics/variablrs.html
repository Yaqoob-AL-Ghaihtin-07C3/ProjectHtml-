مفهوم المتغيرات

المتغيرات (Variables) في CSS هي طريقة لتعريف القيم بشكل يسهّل استعمالها والتعديل عليها لاحقاً، مما يقلل من التكرار ويسهل إدارة التصميم.

الهدف الأساسي من تعريف المتغير هو أن يمثل قيمة ستستخدم بشكل متكرر في تصميم الصفحة.

طريقة تعريف متغير عام

يتم تعريف المتغيرات العامة بداخل المحدد :root { } لتكون متاحة على مستوى الصفحة بأكملها. تبدأ أسماء المتغيرات دائماً برمزين --.

المثال الأول (متغيرات الألوان)

:root {
    --primary: #0275d8;
    --danger: #dc3545;
}
.send {
    background-color: var(--primary);
}
.delete {
    background-color: var(--danger);
}

المثال الثالث (متغيرات المدة الزمنية)

:root {
    --slow: 1s;
}
input {
    transition: all ease-in-out var(--slow);
}

طريقة تعريف المتغير خاص

يمكنك تعريف المتغير بشكل مباشر داخل العنصر ليصبح متاحاً للاستخدام فيه فقط.

المثال الأول (متغير محلي)

a {
    --primary: crimson;
    color: var(--primary);
}
رابط أحمر

المتغيرات مع إستعلامات الوسائط

يمكنك استخدام المتغيرات مع استعلامات الوسائط (Media Queries) لتغيير القيم بناءً على حجم الشاشة.

مثال

:root {
    --font-x1: 20px;
    ...
}
h1 {
    font-size: var(--font-x4);
}
@media (max-width: 976px) {
    h1 {
        font-size: var(--font-x3);
    }
}
