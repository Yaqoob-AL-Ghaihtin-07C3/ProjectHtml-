مفهوم الأنيميشين

الأنيميشين (Animations) هي مؤثرات بصرية تلقائية تجعل خصائص العنصر تتغير تدريجياً عبر الزمن، مما يضيف حيوية وجاذبية لصفحة الويب.

أساسيات الأنيميشين

لإضافة أنيميشين، يجب عليك: تعريف الأنيميشين باستخدام @keyframes، ثم تطبيقه على العنصر باستخدام animation-name و animation-duration.

مثال

@keyframes example {
    from { background-color: red; }
    to { background-color: blue; }
}
div {
    animation-name: example;
    animation-duration: 3s;
}
نص

المحددات المئوية للأنيميشين

يمكنك استخدام النسب المئوية (مثل 0%، 50%، 100%) داخل @keyframes لتحديد مراحل متعددة للتغيير.

مثال

@keyframes example {
    0% { background-color: orange; }
    33% { background-color: red; }
    ...
}
نص

تأخير موعد بدء الأنيميشين

الخاصية animation-delay تحدد مدة التأخير قبل بدء الأنيميشين.

المثال الأول (تأخير إيجابي)

animation-delay: 2s;
تأخير 2s

تحديد كم مرة سيتكرر الأنيميشين

الخاصية animation-iteration-count تحدد عدد مرات تكرار الأنيميشين. يمكنك استخدام عدد محدد أو القيمة infinite للتكرار اللانهائي.

المثال الثاني (تكرار لا نهائي)

animation-iteration-count: infinite;
تكرار

تحديد إتجاه تشغيل الأنيميشين

الخاصية animation-direction تتحكم في اتجاه تشغيل الأنيميشين، سواء كان عادياً، عكسياً، أو متناوباً.

المثال الثاني (متناوب)

animation-direction: alternate;
animation-iteration-count: infinite;
متناوب

تحديد منحنى سرعة الأنيميشين

الخاصية animation-timing-function تتحكم في سرعة الأنيميشين خلال دورته.

مثال

لاحظ الفرق بين منحنيات السرعة المختلفة:

linear
ease
ease-in
ease-out
ease-in-out

إيقاف | تشغيل الأنيميشين

الخاصية animation-play-state تسمح لك بإيقاف (paused) أو تشغيل (running) الأنيميشين.

مثال

div:hover {
    animation-play-state: paused;
}
مرر لإيقاف

إضافة أنيميشين بشكل مختصر

يمكنك دمج جميع خصائص الأنيميشين في خاصية واحدة وهي animation.

المثال الأول

div {
    animation: 2s ease-in-out infinite alternate example;
}
اختصار

إضافة أنيميشين لصورة SVG

يمكنك تحريك رسومات SVG عن طريق إضافتها مباشرة ككود في الصفحة ثم تطبيق الأنماط عليها.

مثال

svg {
    animation: logo-slide 1.5s ease-in-out infinite alternate;
}
#sticker {
    animation: sticker-rotate 1.5s ease-in-out infinite alternate;
}
