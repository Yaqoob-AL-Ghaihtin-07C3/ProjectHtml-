مفهوم وضع صورة كحدود

بدلاً من إظهار حدود تقليدية للعناصر، يمكنك عرض صور جميلة جداً كحدود مع إمكانية تحديد كيف سيتم تقطيعها وعرضها بالشكل الذي تريده. 

border-image: url(border.png) 30;
border-image: url(border.png) 20% round;
border-image: url(border.png) 30 round;

فهم آلية تقسيم الصورة المستخدمة كحدود

الصورة التي سنستخدمها كحدود في هذا الدرس هي التالية و لاحظ كيف أن أحجام المربعات فيها متساوية.

قياسات صورة الحدود

حجم هذه الصورة هو 90 × 90 وهذا الحجم ليس موضوع بشكل عشوائي، بل اخترناه ليكون التعامل مع الصورة سهل جداً حيث أنه عند تقسيمه على ثلاثة سيعطينا 30 وهذا يعني أن كل مربع يظهر في الصورة حجمه 30 بكسل.

طريقة وضع صورة كحدود

أول خطوة هي تحديد سمك الحدود باستخدام الخاصية border. بعدها نستخدم الخاصية border-image-source لتحديد مسار الصورة، وأخيراً نستخدم border-image-slice لتحديد طريقة تقسيم الصورة.

مثال

.border-img {
    border: 10px solid cornflowerblue;
    border-image-source: url(border.png);
    border-image-slice: 30;
}

تحديد كيفية ظهور الأجزاء الداخلية لصورة الحدود

تُستخدم الخاصية border-image-repeat لتحديد كيفية تكرار أو تمديد أجزاء الصورة الداخلية. يمكن أن تأخذ قيمًا مثل stretch، repeat، round، أو space.

المثال الأول (تكرار الأجزاء)

.border-img-stretch { border-image-repeat: stretch; }
.border-img-repeat { border-image-repeat: repeat; }
.border-img-round { border-image-repeat: round; }
.border-img-space { border-image-repeat: space; }

تحديد حجم حدود الصورة الموضوعة كحدود

تُستخدم الخاصية border-image-width لتحديد سمك الصورة التي تُستخدم كحدود.

المثال الأول (قيمة واحدة)

.border-img {
    padding: 20px;
    border-image-width: 10px;
}

طريقة وضع صورة كحدود خارجية

يمكنك استخدام الخاصية border-image-outset لجعل حدود الصورة تظهر خارج حدود العنصر.

مثال

.border-img {
    border-image-outset: 30px;
}

طريقة وضع ألوان متدرجة كحدود للعنصر

يمكن استخدام الدالة `repeating-linear-gradient()` لإنشاء حدود بألوان متدرجة.

مثال

.border-gradient {
    border-image-source: repeating-linear-gradient(45deg, pink, dodgerblue 20px);
}

تعريف كل خصائص الصورة الموضوعة كحدود بسطر واحد

الخاصية border-image هي اختصار لجميع خصائص حدود الصورة، حيث يمكنك دمجها في سطر واحد.

مثال

.border-img-1 { border-image: url(border.png) 30 stretch; }
.border-img-2 { border-image: url(border.png) 30 round; }
.border-img-3 { border-image: url(border.png) 30 round; }
