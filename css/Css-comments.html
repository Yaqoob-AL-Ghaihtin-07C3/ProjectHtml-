مفهوم التعليقات

التعليق (Comment) هو نص يوضع في الكود كملاحظة للمبرمج. إنه يساعد في شرح أجزاء الكود أو تذكيرك بسبب كتابته، ولا يظهر للمستخدم في الصفحة النهائية.

إضافة تعليق في CSS

يتم وضع التعليق في CSS بين /* ... */. يمكن أن يكون التعليق على سطر واحد أو عدة أسطر.

مثال 1: تعليق على سطر واحد

التعليق يسبق القاعدة التي يصفها.

/* أي فقرة موضوعة في الصفحة ستظهر بلون أحمر */
p {
    color: red;
}

هذا النص سيظهر باللون الأحمر.

مثال 2: تعليق بجانب الكود

التعليق يأتي بعد الخاصية التي يصفها.

p {
    color: blue; /* هذه الخاصية تجعل لون النص أزرق */
}

هذا النص سيظهر باللون الأزرق.

مثال 3: تعليق على عدة أسطر

عندما يكون التعليق طويلاً.

/* أي فقرة موضوعة في
الصفحة ستظهر بلون أخضر */
p {
    color: green;
}

هذا النص سيظهر باللون الأخضر.

الفرق بين تعليقات HTML و تعليقات CSS

الهدف من التعليقات في كلا اللغتين هو نفسه، ولكن طريقة كتابتها ومكان استخدامها يختلفان.

  • تعليقات HTML: تُكتب بين <!-- ... --> ويمكن استخدامها في أي مكان داخل ملف HTML باستثناء داخل وسم <style>.
  • تعليقات CSS: تُكتب بين /* ... */ وتستخدم فقط داخل وسم <style> في ملف HTML أو داخل ملف CSS خارجي.

مثال: استخدام كلا النوعين في نفس الصفحة

<!-- تعليق بلغة HTML -->
<style>
    p {
        color: red; /* تعليق بلغة CSS */
    }
</style>
