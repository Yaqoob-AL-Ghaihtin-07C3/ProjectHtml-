ما هو تأثير الظل

أي عنصر تضيفه في الصفحة يمكنك إظهار تأثير الظل حول محيطه الخارجي، تأثير الظل حول محيطه من الداخل، أما المحتوى النصي فيمكنك إظهار تأثير الظل حول أحرفه. وتستطيع أيضاً إضافة عدة طبقات من الظل فوق بعضها لتحصل على تدرج رائع في الظلال.

CSS Text Shadow Effect

CSS Box Shadow Effect

الظل في عالم الويب هو مجرد خط له حجم ولون ولكن الفرق الأساسي بينه وبينه الحدود العادية على سبيل المثال أنه لا يتم حسابه ضمن حساب حجم العنصر فهو يظهر خلف العنصر مباشرةً.

إضافة ظل للنص

لإضافة ظل حول النص نستخدم الخاصية text-shadow ونمرر لها القيم الخاصة بالظل كإحداثيات ظهوره، لونه وحجم ضبابيته. افتراضياً، لون الظل الذي يتم وضعه للنص هو لون النص نفسه.

مثال 1: ظل بسيط

قمنا بإضافة ظل للنص وجعلناه يبتعد عنه بمقدار 2 بكسل من ناحية اليسار و 2 بكسل من ناحية الأعلى ولونه رمادي باهت.

h1 {
    text-shadow: 2px 2px lightgray;
}

CSS Text Shadow Effect

مثال 2: ظل مع ضبابية

قمنا بإضافة ظل للنص وجعلناه يبتعد عنه بمقدار 2 بكسل من ناحية اليسار و 2 بكسل من ناحية الأعلى، وجعلناه ضبابياً بنسبة 5 بكسل ولونه أزرق.

h1 {
    text-shadow: 2px 2px 5px blue;
}

CSS Text Shadow Effect

مثال 3: تأثير الإضاءة (Neon Effect)

ظل لونه أزرق سماوي (cyan) بدون إزاحة (إحداثياته صفر) وضبابية بنسبة 3 بكسل لكي يظهر وكأنه مضيء.

h1 {
    text-shadow: 0 0 3px cyan;
}

CSS Text Shadow Effect

مثال 4: نص أبيض بظل أسود

نص بلون أبيض مع ظل أسود بدون إزاحة وضبابية 5 بكسل.

h1 {
    color: white;
    text-shadow: 0 0 5px black;
}

CSS Text Shadow Effect

إضافة أكثر من ظل للنص

لوضع أكثر من ظل للنص، قم بوضع فاصلة بين كل ظلّين. الظل الذي تضعه أولاً يظهر فوق الظل الذي يليه.

مثال 1: طبقات ظل متداخلة

ظل أزرق فوقه ظل أحمر أصغر حجماً.

h1 {
    text-shadow: 0 0 15px blue, 0 0 5px red;
}

CSS Multiple Text Shadow Effect

مثال 2: طبقات ظل معكوسة

ظل أحمر فوقه ظل أزرق أكبر حجماً.

h1 {
    text-shadow: 0 0 5px red, 0 0 15px blue;
}

CSS Multiple Text Shadow Effect

مثال 3: الظل كحدود للنص

أربع ظلال بنفس الحجم على كل جهة من النص لجعله يظهر كحدود.

h1 {
    color: orange;
    text-shadow: 0 1px black, -1px 0 black, 1px 0 black, 0 -1px black;
}

CSS Multiple Text Shadow Effect

إضافة ظل للعنصر

لإضافة ظل حول العنصر نفسه نستخدم الخاصية box-shadow ونمرر لها القيم الخاصة بالظل كإحداثيات ظهوره، لونه، حجم الظل نفسه ومقدار ضبابيته.

مثال 1: ظل بسيط للعنصر

ظل يبتعد عن العنصر بمقدار 5 بكسل من ناحية اليسار و 5 بكسل من ناحية الأعلى.

div {
    box-shadow: 5px 5px;
    background-color: #F1F1F1;
}
Box Shadow Effect

مثال 2: ظل بلون محدد

نفس المثال السابق ولكن بلون رمادي فاتح (#cacaca).

div {
    box-shadow: 5px 5px #cacaca;
    background-color: #F1F1F1;
}
Box Shadow Effect

مثال 3: ظل مع ضبابية

ظل يبتعد بمقدار 4 بكسل مع ضبابية 5 بكسل ولون أسود خفيف.

div {
    box-shadow: 4px 4px 5px #cacaca;
    background-color: #F1F1F1;
}
Box Shadow Effect

مثال 4: ظل مع امتداد (Spread)

ظل يبتعد بمقدار 4 بكسل مع ضبابية 5 بكسل وامتداد 10 بكسل زيادة عن حجم العنصر.

div {
    box-shadow: 4px 4px 5px 10px #cacaca;
    background-color: #F1F1F1;
}
Box Shadow Effect

مثال 5: ظل داخلي (Inset)

ظل بدون إزاحة وضبابية 5 بكسل وامتداد 10 بكسل، مع إضافة الخاصية inset لجعله يظهر بداخل العنصر.

div {
    box-shadow: inset 0 0 5px 10px #cacaca;
    background-color: #F1F1F1;
}
Box Shadow Effect

إضافة أكثر من ظل للعنصر

لوضع أكثر من ظل للعنصر، قم بوضع فاصلة بين كل ظلّين. الظل الذي تضعه أولاً يظهر فوق الظل الذي يليه.

مثال 1: ظلان حول العنصر

أضفنا ظلين حول العنصر بلونين مختلفين وإزاحات مختلفة.

div {
    box-shadow: 10px 10px #6d588e, 20px 20px #405378;
}
Multiple Box Shadow Effect

مثال 2: ظلال مع حدود مائلة (border-radius)

جعلنا حدود العنصر مائلة مما أدى لجعل الظلال تظهر بشكل مائل أيضاً.

div {
    border-radius: 10px;
    box-shadow: 10px 10px #6d588e, 20px 20px #405378;
}
Multiple Box Shadow Effect

مثال 3: ظلال داخلية متعددة

أضفنا 5 ظلال داخلية بأحجام مختلفة لتكوين تأثير متدرج.

div {
    box-shadow:
        inset 0 0 0 10px #ee6352,
        inset 0 0 0 20px #d45379,
        inset 0 0 0 30px #a4558f,
        inset 0 0 0 40px #6d588e,
        inset 0 0 0 50px #405378;
}
Box Shadow Effect
