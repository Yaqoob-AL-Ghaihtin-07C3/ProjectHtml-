مفهوم المحتوى الفائض

**المحتوى الفائض** أو الزائد هو المحتوى الذي يتطلب عرضه كاملاً مساحة أكبر من مساحة العنصر الموضوع فيه.

حين يكون المحتوى بحاجة لمساحة أكبر، يظهر عادةً شريط تمرير (Scroll Bar) يسمح لك بالتحرك داخل العنصر حتى تتمكن من مشاهدة المحتوى بالكامل.

مشاكل المحتوى الفائض

بشكل افتراضي، إذا لم تحدد طريقة عرض المحتوى، فإنه سيظهر خارج حدود العنصر. هذا يمكن أن يسبب مشكلة تداخل المحتوى، حيث يظهر العنصر التالي في الصفحة فوق المحتوى الفائض بدلاً من بعده.

إظهار شريط تمرير

يجب أن يكون حجم العنصر محدداً ويظهر كبلوك (سواء `block` أو `inline-block`) لكي تتمكن من إظهار شريط تمرير فيه.

نستخدم الخاصية **`overflow`** لتحديد كيفية التعامل مع المحتوى الفائض، وهي تقبل القيم التالية:

القيمة المعنى
`visible` لجعل المحتوى الفائض يظهر دائماً (القيمة الافتراضية).
`hidden` لجعل المحتوى الفائض مخفياً تماماً، دون إظهار أي شريط تمرير.
`scroll` لإظهار أشرطة التمرير طوال الوقت، حتى لو لم يكن المحتوى بحاجة لها.
`auto` لإظهار أشرطة التمرير تلقائياً فقط إذا كان حجم المحتوى أكبر من حجم العنصر.

مثال 1: `overflow: visible;`

جعلنا المحتوى الفائض ظاهراً.

div {
    overflow: visible;
}

مثال 2: `overflow: hidden;`

قمنا بإخفاء المحتوى الفائض.

div {
    overflow: hidden;
}

مثال 3: `overflow: scroll;`

قمنا بإظهار أشرطة التمرير طوال الوقت.

div {
    overflow: scroll;
}

مثال 4: `overflow: auto;`

جعلنا أشرطة التمرير تظهر عند الحاجة فقط.

div {
    overflow: auto;
}

إظهار شريط التمرير في مكان محدد

يمكنك التحكم في ظهور شريط التمرير بشكل أفقي أو عمودي باستخدام الخصائص التالية:

  • `overflow-x`: للتعامل مع المحتوى الفائض أفقياً.
  • `overflow-y`: للتعامل مع المحتوى الفائض عمودياً.

**ملاحظة:** اتجاه شريط التمرير العمودي يظهر دائماً في الجهة المعاكسة لاتجاه المحتوى (`ltr` أو `rtl`).

مثال: إظهار شريط تمرير عمودي فقط

قمنا بإخفاء شريط التمرير الأفقي وإظهار شريط التمرير العمودي.

div {
    overflow-x: hidden;
    overflow-y: scroll;
}

تقطيع النص الطويل الفائض

في حال وجود نص طويل جداً بدون فواصل، فإنه قد يمتد خارج حدود العنصر. لحل هذه المشكلة، استخدم الخاصية **`overflow-wrap`**.

الخاصية `overflow-wrap` تقبل القيم التالية:

  • `normal`: (القيمة الافتراضية) لا يتم قطع النص.
  • `break-word`: يتم قطع الكلمة الطويلة جداً لكي تنزل على سطر جديد.

مثال: قطع النص الطويل

div {
    overflow-wrap: break-word;
}
