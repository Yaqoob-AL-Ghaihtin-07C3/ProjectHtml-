الشكل العام لكود CSS

كود CSS يستخدم لتحسين مظهر عناصر HTML. عند كتابة الكود، يجب تحديد العنصر المراد تعديله ثم إضافة الخصائص والقيم المناسبة له.

مثال على الشكل العام

h1 {
    color: blue;
    text-align: center;
}

h1: يسمى المُحدّد (Selector)، وهو اسم الوسم الذي سيتم تطبيق الأنماط عليه.

color: تسمى الخاصية (Property)، وهي السمة المراد تعديلها.

blue: تسمى القيمة (Value)، وهي القيمة المحددة للخاصية.

يجب وضع فاصلة منقوطة (;) بعد كل خاصية وقيمتها.

أين تكتب كود CSS

يوجد ثلاثة أساليب يمكنك اتباعها لإضافة كود CSS في صفحات الويب:

1- أسلوب Inline CSS

يتم إضافة الخاصية style مباشرةً إلى وسم HTML وتمرير كود CSS كقيمة لها. 

<h1 style="color:blue; text-align:center;">Hello World!</h1>

Hello World!

2- أسلوب Internal CSS

يتم وضع كود CSS داخل وسم <style> في أي مكان من الصفحة، عادةً في قسم <head>. 

<style>
    h1 {
        color: blue;
        text-align: center;
    } 
</style>

Hello World!

3- أسلوب External CSS

يتم وضع كود CSS في ملف منفصل بامتداد .css، ومن ثم يتم ربطه بالصفحة باستخدام وسم <link>. 

style.css
h1 {
    color: blue;
    text-align: center;
}

index.html
<link rel="stylesheet" href="style.css">

Hello World!

أفضل طريقة لكتابة كود CSS

بشكل عام، أفضل طريقة هي استخدام أسلوب External CSS، حيث يتم تخصيص ملف .css واحد لكل أنماط الموقع، مما يسهل إعادة استخدامه وتعديله. الأساليب الأخرى (Inline و Internal) تستخدم عادةً للتعديلات البسيطة أو المخصصة لعنصر معين لا تتكرر في باقي الصفحات.
