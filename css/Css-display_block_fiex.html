مفهوم أساليب العرض

أي عنصر تضيفه في الصفحة يظهر بأسلوب معين، فإما أن يظهر على السطر بأكمله ولا يظهر شيء آخر بجانبه، وإما أنه يظهر بجانب عناصر أخرى في نفس السطر إذا كان هناك مساحة تسمح بذلك.

في هذا الدرس ستتعرف على أساليب العرض الرئيسية التي تملكها عناصر HTML وكيف يمكنك تغييرها بواسطة خصائص CSS لتلائم تصاميمك.

أساليب عرض العناصر

لتحديد أسلوب العرض نستخدم الخاصية display ومن بعدها نمرر لها قيمة تمثل أسلوب العرض الذي نريد أن يعتمده العنصر.

الأساليب الأساسية

الأساليب الأساسية التي تعتمدها وسوم HTML والتي يمكن تمريرها كقيمة لها هي:

  • block: لجعل العنصر يظهر على كامل السطر (إن لم تتلاعب بحجمه) ولا يظهر أي شيء آخر بجانبه.
  • inline-block: لجعل العنصر يظهر كبلوك ولا مانع أن يظهر عنصر آخر بجانبه إن كان هناك متسع له.
  • inline: لجعل العنصر يظهر بنفس حجم محتواه وهذا الأسلوب في العرض يستخدم بشكل خاص عندما تحتاج إلى تعديل تصميم نص موجود في فقرة أو عنوان.

خصائص الأساليب

العنصر الذي يظهر كبلوك (سواء block أو inline-block) يمكن إضافة padding و margin له من كل الجهات، وكذلك يمكن تحديد حجمه بواسطة خاصيتي width و height.

أما العنصر الذي يظهر بأسلوب inline فلا يمكن تحديد حجمه. وفي حال إضافة هامش داخلي أو خارجي له من الأعلى والأسفل، لن يظهر أي تغيير، وهذا أمر منطقي لأن هذا الأسلوب يُستعمل بالأساس لتطبيق تصميم معين على كلام معين في نص.

أمثلة عملية

مثال 1: جعل زر يظهر كـ block

في هذا المثال، قمنا بجعل الوسم <button> يعتمد أسلوب block حتى يظهر لوحده على السطر.

button {
    display: block;
}

مثال 2: جعل div يظهر كـ inline-block

في هذا المثال، قمنا بجعل الوسم <div> يعتمد أسلوب inline-block حتى لا يظهر على سطر لوحده.

div {
    display: inline-block;
}
محتوى البطاقة

الظهور الافتراضي للعناصر الجديدة

أي عنصر جديد تقوم بتعريفه بنفسك يستخدم أسلوب inline بشكل تلقائي ما لم تقم أنت بالتلاعب بطريقة ظهوره.

في هذا المثال، قمنا بإضافة وسم جديد اسمه <harmash> الذي يعتمد أسلوب inline للظهور في الصفحة، وجعلناه يظهر بلون أحمر وأضفنا له هامش وحشو وإطار.

ملاحظة: أثناء تجربة الكود، قم بتغيير قيمة الخاصية display من inline إلى inline-block ومن ثم إلى block لترى الفرق. 

harmash {
    display: inline;
    color: red;
    margin: 30px;
    padding: 30px;
    border: 1px solid red;
}

أساليب إخفاء العناصر

يوجد أسلوبان أساسيان لإخفاء العناصر:

display: none

هذا الأسلوب يزيل العنصر تماماً من الصفحة وكأنه لم يعد موجوداً فيها. لا يتم حجز أي مساحة له.

h1 {
    display: none;
}

visibility: hidden

هذا الأسلوب يخفي العنصر من الصفحة ولكن مكانه يظل محجوزاً فيها. أي أن المساحة التي كان يشغلها تبقى فارغة.

h1 {
    visibility: hidden;
}
