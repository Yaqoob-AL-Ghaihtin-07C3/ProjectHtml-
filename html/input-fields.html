مفهوم حقول الإدخال

حقول الإدخال أو مربعات النصوص في صفحات الويب يمكن تصنيفها لنوعين أساسيين:

مربع نص عادي

سطر واحد للإدخال

<input type="text">
مربع نص متعدد

عدة أسطر للإدخال

<textarea></textarea>

الوسوم الأساسية

<input> - لمربعات النص ذات السطر الواحد
<textarea> - لمربعات النص متعددة الأسطر

إضافة مربع نص

لإضافة مربع نص في الصفحة يمكنك استخدام الوسم <input>.

خصائص الوسم <input>

type - نوع الحقل (text, password, email, number)
id - معرف فريد للربط مع <label>
name - اسم الحقل لإرسال البيانات
value - قيمة افتراضية للحقل
placeholder - تلميح يظهر عند الفراغ
required - يجعل الحقل إلزامياً
readonly - للقراءة فقط
disabled - تعطيل الحقل
size - عرض الحقل بعدد الأحرف
maxlength - أقصى عدد للأحرف

نص عادي

<label for="username">اسم المستخدم</label>
<input type="text" id="username" name="username">

كلمة المرور

<label for="password">كلمة المرور</label>
<input type="password" id="password" name="password">

البريد الإلكتروني

<label for="email">البريد الإلكتروني</label>
<input type="email" id="email" name="email">

الأرقام

<label for="age">العمر</label>
<input type="number" id="age" name="age">

مثال: خصائص متقدمة

<label for="user">اسم المستخدم</label>
<input type="text" id="user" name="user" 
       size="20" 
       maxlength="25" 
       placeholder="ادخل اسم المستخدم" 
       required
>

إضافة مربع نص متعدد الأسطر

نستخدم الوسم <textarea> للنص متعدد الأسطر.

خصائص الوسم <textarea>

id - معرف فريد للربط مع <label>
name - اسم الحقل لإرسال البيانات
rows - عدد الأسطر الظاهرة
cols - عدد الأعمدة (العرض)
readonly - للقراءة فقط
disabled - تعطيل الحقل
placeholder - تلميح يظهر عند الفراغ

نص متعدد أساسي

<label for="bio">السيرة الذاتية</label>
<textarea id="bio" name="bio">
    النص الافتراضي هنا
</textarea>

مع تحديد الحجم

<label for="message">الرسالة</label>
<textarea id="message" name="message" 
          rows="4" 
          cols="50"
></textarea>

حالات خاصة للحقول

قيمة افتراضية 

<input type="text" value="القيمة الافتراضية">

للقراءة فقط 

<input type="text" value="قيمة للقراءة" readonly>

معطل 

<input type="text" value="قيمة معطلة" disabled>
الحالة الوصف المظهر التفاعل عادي الحقل العادي طبيعي كامل التفاعل readonly للقراءة فقط خلفية رمادية يمكن التركيز ولكن لا يمكن التعديل disabled معطل خلفية رمادية فاتحة لا يمكن التركيز أو التعديل
ملاحظة: الحقول المعطلة (disabled) لا يتم إرسال قيمها عند تقديم النموذج، بينما الحقول للقراءة فقط (readonly) يتم إرسال قيمها.
