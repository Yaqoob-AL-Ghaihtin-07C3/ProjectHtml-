مفهوم الإقتباس

النص المقتبس (Quoted text) يقصد به أن النص الموضوع في الصفحة تم نقله من مصدر آخر.

عادةً ما يتم عرض النص المقتبس بشكل مختلف قليلاً عن باقي النص الموجود في الصفحة بهدف إعلام القارئ بأنه منقول من مصدر آخر و لحفظ حقوق المصدر الذي نقل الكلام منه.

في هذا الدرس سنتعرف على الوسوم المخصصة لعرض إقتباسات في الصفحة.

عرض النص المقتبس بداخل بلوك

إذا أردت العرض النص مقتبس على سطر منفرد مع عرض هامش حوله من كل الجهات، يمكنك وضع النص بداخل <blockquote></blockquote>.

المثال الأول: إقتباس بدون مصدر

<blockquote>
In 1980, physicist Tim Berners-Lee, a contractor at CERN, proposed and prototyped 
ENQUIRE, a system for CERN researchers to use and share documents. In 1989, Berners-Lee 
wrote a memo proposing an Internet-based hypertext system.
</blockquote>
In 1980, physicist Tim Berners-Lee, a contractor at CERN, proposed and prototyped ENQUIRE, a system for CERN researchers to use and share documents. In 1989, Berners-Lee wrote a memo proposing an Internet-based hypertext system.

المثال الثاني: إقتباس مع مصدر

إذا كان الإقتباس الموضوع في الصفحة منسوخ من موقع آخر، قم بذكر رابط الموقع الذي نسخت الإقتباس منه بداخل الخاصية cite.

<blockquote cite="https://en.wikipedia.org/wiki/HTML">
In 1980, physicist Tim Berners-Lee, a contractor at CERN, proposed and prototyped 
ENQUIRE, a system for CERN researchers to use and share documents. In 1989, Berners-Lee 
wrote a memo proposing an Internet-based hypertext system.
</blockquote>
In 1980, physicist Tim Berners-Lee, a contractor at CERN, proposed and prototyped ENQUIRE, a system for CERN researchers to use and share documents. In 1989, Berners-Lee wrote a memo proposing an Internet-based hypertext system.

عرض النص المقتبس بين " "

إذا أردت عرض النص المقتبس بين " " يمكنك وضعه بداخل <q></q>.

إستخدام الوسم <q> بدلاً من كتابة " " بشكل مباشر يُعد خيار أفضل حيث أنه يمكن تخصيص طريقة عرض الوسم بواسطة CSS.

المثال الأول: إقتباس مضمن

<p>My teacher said: <q>HTML is not a programming language</q></p>

My teacher said: HTML is not a programming language

المثال الثاني: تخصيص الإقتباس باستخدام CSS

<style>
q { 
    color: red;
    quotes: "“" "”" "‘" "’";
}
</style>

<p>My teacher said: <q>HTML is not a programming language</q>
and he also said that <q>It's very easy to learn</q></p>

My teacher said: HTML is not a programming language and he also said that It's very easy to learn

وضع إسم مختصر بدل الإسم الكامل

في البداية ما نقصده بإسم مختصر (Abbreviation) هو أنك على سبيل المثال تستطيع عرض كلمة واحدة مثل HTML و عند تمرير الفأرة فوقها يقوم المتصفح بإظهار الإسم الكامل Hyper Text Markup Language.

إذا أردت وضع إسم مختصر فيمكنك وضعه بداخل <abbr></abbr>.

مثال: استخدام الاختصارات

<p><abbr title="Hyper Text Markup Language">HTML</abbr> is used to create web pages.</p>
<p><abbr title="Cascading Style Sheets">CSS</abbr> is used to style web pages.</p>
<p><abbr title="JavaScript">JS</abbr> is used to add interactivity to web pages.</p>

HTML is used to create web pages.

CSS is used to style web pages.

JS is used to add interactivity to web pages.

عرض معلومات الوصول و التواصل

إذا أردت عرض معلومات الوصول أو التواصل فيمكنك وضع النص بداخل <address></address>.

أنواع المعلومات

  • معلومات الوصول: الموقع الجغرافي مثل "مكتب سالم المصري, شارع الإصلاح، مبنى السلام، الطابق الثاني"
  • معلومات التواصل: رقم الهاتف، البريد الإلكتروني، صندوق البريد إلخ

مثال: معلومات التواصل والوصول

<address>
9645 Al-islah street, Tripoli, 1300 - Lebanon
</address>

<address>
هاتف: 061584984
إيميل: info@harmash.com
صندوق البريد: 12345
</address>
9645 Al-islah street, Tripoli, 1300 - Lebanon
هاتف: 061584984
إيميل: info@harmash.com
صندوق البريد: 12345

إبراز عنوان العمل

في حال أردت إبراز عنوان الشيء المهم الذي يتم عرضه يمكنك وضعه بداخل <cite></cite>.

مثال: عناوين الأعمال

<p><cite>Riyadh-as-Saliheen</cite> by Imam Al-Nawawi, volume 1, hadith number 134</p>

<p>One of the most famous books in programming is <cite>Clean Code</cite> by Robert C. Martin.</p>

<p>The movie <cite>The Shawshank Redemption</cite> is considered one of the best films of all time.</p>

Riyadh-as-Saliheen by Imam Al-Nawawi, volume 1, hadith number 134

One of the most famous books in programming is Clean Code by Robert C. Martin.

The movie The Shawshank Redemption is considered one of the best films of all time.
