أنواع الأزرار

هناك 4 أنواع رئيسية من الأزرار التي يمكن إضافتها في النماذج:

أنواع أزرار الإدخال

type="button" - زر عادي
type="submit" - زر إرسال البيانات
type="reset" - زر إعادة التعيين
type="file" - زر اختيار الملف
النوع الوصف الاستخدام القيمة الافتراضية button زر عادي أي غرض عام يحددها المطور submit زر إرسال إرسال النموذج "Submit" reset زر إعادة تعيين مسح بيانات النموذج "Reset" file زر ملف رفع الملفات "Choose File"

إضافة زر عادي

يمكن استخدام الوسم <button> أو <input type="button">

使用 <button>

<button type="button">زر عادي</button>

使用 <input>

<input type="button" value="زر عادي">

مثال: أزرار مع JavaScript

<button type="button" onclick="alert('مرحباً!')">
    عرض رسالة
</button>

<input type="button" value="عرض رسالة" onclick="alert('مرحباً!')">

مثال: زر بأيقونة

<button type="button" class="icon-button">
    <img src="play-icon.png" alt="Play">
    تشغيل
</button>

إضافة زر لإرسال بيانات النموذج

يستخدم <input type="submit"> لإرسال بيانات النموذج

زر الإرسال

<input type="submit" value="تسجيل الدخول">
ملاحظة: الزر submit يرسل بيانات النموذج تلقائياً إلى العنوان المحدد في خاصية action للوسم <form>.

إضافة زر لإعادة تعيين قيم النموذج

يستخدم <input type="reset"> لإعادة تعيين النموذج

زر إعادة التعيين

<input type="reset" value="إعادة التعيين">
ملاحظة: الزر reset يعيد جميع حقول النموذج إلى قيمها الافتراضية الأصلية.

إضافة زر لاختيار الملفات

يستخدم <input type="file"> لاختيار الملفات

زر اختيار الملف

<input type="file">

<!-- للسماح بملفات متعددة -->
<input type="file" multiple>

<!-- لتحديد نوع الملف -->
<input type="file" accept=".jpg,.png,.pdf">

خصائص زر الملف

multiple - السماح باختيار multiple files
accept - تحديد أنواع الملفات المقبولة
capture - فتح الكاميرا مباشرة (على الهواتف)

نصائح مهمة لاستخدام الأزرار

  • ✅ استخدام <button> عندما تحتاج إلى محتوى معقد داخل الزر
  • ✅ استخدام <input> للأزرار البسيطة
  • ✅ تحديد type دائمًا للأزرار
  • ✅ استخدام نصوص واضحة ومعبرة على الأزرار
  • ✅ توفير أزرار واضحة للنماذج (Submit, Reset)
  • ❌ تجنب استخدام الأزرار المعطلة دون سبب واضح
❮ حقول الإدخال خانات الاختيار ❯
