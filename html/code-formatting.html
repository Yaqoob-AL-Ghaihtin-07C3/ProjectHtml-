مفهوم تنسيق الكود

تنسيق الكود نقصد به إظهار النص الذي يرمز لكود برمجي، مدخلات و مخرجات الحاسوب أو لعملية حسابية ما بنوع خط مختلف عن النص العادي الموجود في الصفحة.

نوع الخط الذي يستخدم في تنسيق الكود إسمه monospace و هو يعطي الأحرف حجماً موحداً مما يجعل قراءتها و تتبعها أمر سهل جداً بالنسبة للقارئ.

في هذا الدرس ستتعرف على الوسوم المخصصة لتنسيق الكود البرمجي و المعادلات الحسابية.

عرض أسماء أحرف الكيبورد

عند وضع شروحات تخبر فيها المستخدم بأن ينقر على أزرار معينة في الكيبورد (Keyboard) يمكنك وضعها بين <kbd></kbd>.

هذا الوسم يقوم فقط بتغيير نوع خط النص الموضوع بداخله إلى النوع monospace حتى يظهره بشكل مختلف عن باقي النص.

مثال: أزرار الكيبورد

<p>Press <kbd>Ctrl + S</kbd> to save this page on your computer.</p>

Press Ctrl + S to save this page on your computer.

مثال: مجموعة أزرار

<p>Use <kbd>Ctrl + C</kbd> to copy and <kbd>Ctrl + V</kbd> to paste.</p>
<p>Press <kbd>F5</kbd> to refresh the page.</p>
<p>Use <kbd>Alt + Tab</kbd> to switch between applications.</p>

Use Ctrl + C to copy and Ctrl + V to paste.

Press F5 to refresh the page.

Use Alt + Tab to switch between applications.

عرض مخرجات البرامج

في حال أردت عرض مخرجات البرامج (Programs Output) يمكنك وضعها بين <samp></samp>.

هذا الوسم يقوم فقط بتغيير نوع خط النص الموضوع بداخله إلى النوع monospace حتى يظهره بشكل مختلف عن باقي النص.

مثال: رسائل الخطأ

<p>My computer show me this alert:</p>
<p><samp>Error code: 0x80070043<br>The network name cannot be found.</samp></p>

My computer show me this alert:

Error code: 0x80070043
The network name cannot be found.

مثال: مخرجات الأوامر

<p>Output of the command:</p>
<samp>
$ git status
On branch main
Your branch is up to date with 'origin/main'.

nothing to commit, working tree clean
</samp>

Output of the command:

$ git status On branch main Your branch is up to date with 'origin/main'. nothing to commit, working tree clean

عرض كود في النص

في حال أردت عرض الكلام الذي هو عبارة عن كود أو عملية حسابية بشكل مميز عن النص الموضوع معه في نفس الفقرة يمكنك وضع الكود بين <code></code>.

هذا الوسم يقوم فقط بتغيير نوع خط النص الموضوع بداخله إلى النوع monospace حتى يظهره بشكل مختلف عن النص العادي.

المثال الأول: دوال برمجية

<p>In java the <code>main()</code> function is the starting point.</p>
<p>Use <code>console.log()</code> to print messages in JavaScript.</p>

In java the main() function is the starting point.

Use console.log() to print messages in JavaScript.

المثال الثاني: معادلات رياضية

<p>Pythagorean Theorem: <code>a<sup>2</sup> + b<sup>2</sup> = c<sup>2</sup></code></p>
<p>Quadratic formula: <code>x = (-b ± √(b² - 4ac)) / 2a</code></p>

Pythagorean Theorem: a2 + b2 = c2

Quadratic formula: x = (-b ± √(b² - 4ac)) / 2a

المثال الثالث: أكواد قصيرة

<p>CSS property: <code>font-family</code></p>
<p>JavaScript operator: <code>===</code></p>

HTML tag: <div>

CSS property: font-family

JavaScript operator: ===

عرض أسماء المتغيرات في النص

المواضيع التي لها علاقة بالبرمجة و الرياضيات و أي مادة أخرى تتضمن معادلات نقوم عادةً باستخدام أسماء متغيرات فيها مثل X و Y.

إذا أردت عرض إسم المتغير بشكل مميز عن النص الموضوع معه في نفس الفقرة يمكنك وضعه بين <var></var>.

هذا الوسم يقوم بعرض الإسم الذي نضعه بداخله بشكل مائل.

مثال: متغيرات رياضية

<p>Calculate the value of <var>x</var> where <code>x + 5 = 13</code>.</p>
<p>The formula for area of circle is <var>A</var> = π<var>r</var><sup>2</sup>.</p>
<p>Solve for <var>y</var> in the equation <code>2<var>y</var> + 8 = 20</code>.</p>

Calculate the value of x where x + 5 = 13.

The formula for area of circle is A = πr2.

Solve for y in the equation 2y + 8 = 20.

مثال: متغيرات برمجية

<p>Declare a variable <var>userName</var> to store the user's name.</p>
<p>Use <var>counter</var> to keep track of the number of iterations.</p>
<p>The function returns <var>result</var> which contains the final output.</p>

Declare a variable userName to store the user's name.

Use counter to keep track of the number of iterations.

The function returns result which contains the final output.
