مفهوم القوائم الغير مرتبة

القائمة الغير مرتبة (Unordered List) يقصد بها عرض مجموعة عناصر بدون الإهتمام ما إن كان يجب عرض عنصر قبل عنصر آخر فيها.

في هذا النوع من القوائم يمكنك تحديد الرمز الذي سيظهر في بداية كل عنصر و هنا يوجد ثلاث رموز يمكنك عرضها هي مربع، دائرة، أو قرص.

في هذا الدرس ستتعرف على وسوم و خصائص القوائم الغير مرتبة بالإضافة إلى كيفية وضع قائمة بداخل قائمة.

إضافة قائمة غير مرتبة في الصفحة

بشكل عام، نستخدم الوسم <ul></ul> لإعلام المتصفح أننا نريد عرض قائمة غير مرتبة في الصفحة.

أي عنصر نريد إضافته فيها يجب وضعه بداخل الوسم <li></li>.

مثال: قائمة غير مرتبة أساسية

<ul>
    <li>Apple</li>
    <li>Orange</li>
    <li>Mango</li>
</ul>
  • Apple
  • Orange
  • Mango

أنواع الرموز التي يمكن عرضها في أول العناصر

لتحديد نوع الرموز التي ستظهر قبل العناصر يمكنك إضافة الخاصية type للوسم <ul>.

النوع square - مربع

<ul type="square">
    <li>Apple</li>
    <li>Orange</li>
    <li>Mango</li>
</ul>
  • Apple
  • Orange
  • Mango

النوع circle - دائرة

<ul type="circle">
    <li>Apple</li>
    <li>Orange</li>
    <li>Mango</li>
</ul>
  • Apple
  • Orange
  • Mango

النوع disc - قرص (افتراضي)

<ul type="disc">
    <li>Apple</li>
    <li>Orange</li>
    <li>Mango</li>
</ul>
  • Apple
  • Orange
  • Mango

النوع none - بدون رمز

<ul type="none">
    <li>Apple</li>
    <li>Orange</li>
    <li>Mango</li>
</ul>
  • Apple
  • Orange
  • Mango

تغيير نوع الرمز الذي سيظهر قبل عنصر محدد في القائمة

يمكن استخدام الخاصية type على العنصر <li> لتغيير الرمز لعنصر معين.

مثال: تغيير رمز العنصر الثاني فقط

<ul>
    <li>Apple</li>
    <li type="circle">Orange</li>
    <li>Mango</li>
</ul>
  • Apple
  • Orange
  • Mango

مثال: رموز مختلفة لعناصر مختلفة

<ul>
    <li type="disc">Apple (قرص)</li>
    <li type="circle">Orange (دائرة)</li>
    <li type="square">Mango (مربع)</li>
    <li type="none">Banana (بدون رمز)</li>
</ul>
  • Apple (قرص)
  • Orange (دائرة)
  • Mango (مربع)
  • Banana (بدون رمز)

وضع قائمة بداخل قائمة

يمكن إنشاء قوائم متداخلة بوضع قائمة داخل عنصر من القائمة الرئيسية.

مثال: قوائم متداخلة

<ul>
    <!-- قائمة داخل العنصر الأول -->
    <li>Front-End
        <ul>
            <li>HTML</li>
            <li>CSS</li>
            <li>Javascript</li>
        </ul>
    </li>
    
    <!-- قائمة داخل العنصر الثاني -->
    <li>Back-End
        <ul>
            <li>MySQL</li>
            <li>Aspnetcore</li>
        </ul>
    </li>
</ul>
  • Front-End
    • HTML
    • CSS
    • Javascript
  • Back-End
    • MySQL
    • Aspnetcore

مثال: قوائم متداخلة متعددة المستويات

<ul>
    <li>المستوى الأول
        <ul>
            <li>المستوى الثاني
                <ul>
                    <li>المستوى الثالث</li>
                </ul>
            </li>
        </ul>
    </li>
</ul>
  • المستوى الأول
    • المستوى الثاني
      • المستوى الثالث
