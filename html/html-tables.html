مفهوم الجدول

الجدول (Table) يسمح لك بعرض المعلومات بشكل مرتب على شكل أعمدة و أسطر.

في هذا الدرس ستتعلم كيف تنشئ جدول، كيف تضيف فيه أسطر، كيف تضيف فيه أعمدة و كيف تدمج الأعمدة و الأسطر مع بعضها.

إضافة جدول في الصفحة

لعرض جدول في الصفحة نستخدم الوسوم التالية:

  • <table></table> - الوسم الأساسي للجدول
  • <tr></tr> - لإنشاء سطر في الجدول
  • <th></th> - لإنشاء خلية عنوان (تظهر بخط عريض)
  • <td></td> - لإنشاء خلية بيانات عادية

مثال: جدول أساسي

<table border="1">
    <!-- السطر الأول - العناوين -->
    <tr>
        <th>ID</th>
        <th>Name</th>
        <th>Mark</th>
    </tr>
    
    <!-- السطر الثاني -->
    <tr>
        <td>1</td>
        <td>Mhamad Harmush</td>
        <td>A</td>
    </tr>
    
    <!-- السطر الثالث -->
    <tr>
        <td>2</td>
        <td>Rima Houssayni</td>
        <td>C</td>
    </tr>
    
    <!-- السطر الرابع -->
    <tr>
        <td>3</td>
        <td>Youssef Jabber</td>
        <td>B</td>
    </tr>
</table>
ID Name Mark
1 Mhamad Harmush A
2 Rima Houssayni C
3 Youssef Jabber B

خصائص الجدول

هناك ثلاث خصائص أساسية يمكن استخدامها لتنسيق الجدول:

  • border - لإظهار خطوط الجدول
  • width - لتحديد عرض الجدول
  • height - لتحديد طول الجدول

مثال: جدول بخصائص مخصصة

<table border="2" width="100%" height="200">
    <tr>
        <th>الخاصية</th>
        <th>الوصف</th>
        <th>القيمة</th>
    </tr>
    <tr>
        <td>border</td>
        <td>سمك الحدود</td>
        <td>2px</td>
    </tr>
    <tr>
        <td>width</td>
        <td>عرض الجدول</td>
        <td>100%</td>
    </tr>
    <tr>
        <td>height</td>
        <td>طول الجدول</td>
        <td>200px</td>
    </tr>
</table>
الخاصية الوصف القيمة
border سمك الحدود 2px
width عرض الجدول 100%
height طول الجدول 200px

دمج خانات الجدول

يمكن دمج الخلايا باستخدام الخاصيتين:

  • colspan - لدمج الخلايا أفقيًا
  • rowspan - لدمج الخلايا عموديًا

المثال الأول: دمج أفقي (colspan)

<table border="1">
    <tr>
        <td colspan="2">A (مدمج خانتين)</td>
        <td>B</td>
    </tr>
    <tr>
        <td>C</td>
        <td>D</td>
        <td>E</td>
    </tr>
</table>
A (مدمج خانتين) B
C D E

المثال الثاني: دمج عمودي (rowspan)

<table border="1">
    <tr>
        <td>A</td>
        <td rowspan="2">B (مدمج سطرين)</td>
        <td>C</td>
    </tr>
    <tr>
        <td>D</td>
        <td>E</td>
    </tr>
</table>
A B (مدمج سطرين) C
D E

المثال الثالث: دمج مزدوج (colspan + rowspan)

<table border="1">
    <tr>
        <td>A</td>
        <td>B</td>
        <td>C</td>
    </tr>
    <tr>
        <td>D</td>
        <td colspan="2" rowspan="2">E (مدمج خانتين أفقيًا وسطرين عموديًا)</td>
    </tr>
    <tr>
        <td>F</td>
    </tr>
</table>
A B C
D E (مدمج خانتين أفقيًا وسطرين عموديًا)
F

وضع عنوان خاص للجدول

يمكن استخدام الوسم <caption> لإضافة عنوان للجدول.

مثال: جدول بعنوان

<table border="1">
    <caption>علامات الفصل النهائي</caption>
    <tr>
        <th>ID</th>
        <th>Name</th>
        <th>Mark</th>
    </tr>
    <tr>
        <td>1</td>
        <td>Mhamad Harmush</td>
        <td>A</td>
    </tr>
</table>

ID Name Mark
1 Mhamad Harmush A
